> **研究日期 / 数据截止：** 2026-07-16（CST）

## 0. 执行摘要

| 项 | 结论（锚定 7/2–7/4 地图 + 本刊同日整理） |
|----|------|
| 趋势确认 | ✅ 可追踪：物理扩建仍在；但「钱已走过五站」，第一层/多数第二层**充分甚至过度定价** |
| 当前 S 级（物理真紧） | **InP/EML**、**ABF/高阶载板**、**T-glass/Low-Dk 玻纤**、**大型变压器/并网与部分发电侧**、**WF6/氦等特气材料**；7/4 增量含 **HVLP4 铜箔、BT 树脂、柴发、核电大锻件** 等 |
| 已降级 / 非瓶颈 | **CoWoS**（缺口收窄、2027 转松叙事）、**液冷整机/冷板**（供给跟上+价格战）、**光模块整机**（受上游配给，自身非稀缺） |
| 仍紧但已定价 / 慎买 | HBM 芯片层、多数光芯片龙头、Vertiv/Bloom 等电力设备热门股 |
| 错配方向（7/2 口径） | IPP 电力（瓶颈之因、股价偏弱）、MOCVD、HBM4 测试耗材、二线载板/玻纤认证者等——**以 7/2、7/4 机会看板为准，本 L1 不重开股票终选** |
| 是否触发新 L2 | **默认否**（仓库已有 7 月初扫描与机会档）；仅当 L0 出现**新证伪/新涨价落地**或用户指定子集时再开 funnel |
| 主题仓位（观点） | *收缩对「已透支组装层」暴露；真瓶颈优先看材料/电力物理层，且必须过估值门* |

**一句话：** 瓶颈已从「GPU/封装位」迁到 **原料·精密辅材·电力物理层**；5 月稿把 CoWoS/液冷整机当峰值瓶颈的表述，**以 7/2–7/4 为准予以取代**。

**反面：** Meta Compute（7/1）出租闲置算力，是超级买家侧对「电力永远稀缺」的首次明显打脸信号——电力 S 级仍成立于变压器/并网/燃料侧，但「土地/排队占位泡沫」与「物理稀缺」必须分开看（见 7/4）。

## 1. 超级趋势确认

| 标准 | 判断 | 证据锚点 |
|------|------|----------|
| 持续性 | 通过 | 长交期电力设备、材料涨价落地、长约锁料仍见公开报道 |
| 物理性 | 通过 | 高密机柜、光/电/材料同时紧（非纯软件） |
| 规模性 | 通过 | 云与主权算力 capex 仍在；TrendForce 等已下修部分服务器出货增速（减速信号存在） |
| 加速性 | **分化** | S 级材料/电力仍紧；CoWoS/液冷整机供给弹性已跟上 |

**趋势：✅ 可追踪，但进入「晚扩张 + 定价透支」阶段**（与 7/2「Late-Expansion / 钱走完五站」一致）。

## 2. 物理拆解（简）

```
Layer 0  训练/推理服务
Layer 1  GPU / HBM 芯片 / 整机          ← 关注最高，多数已定价
Layer 2  光模块组装、液冷整机、交换机   ← 需求在，稀缺性弱或已透支
──────────────── 当前 alpha 扫描线 ────────────────
Layer 2.5–3  InP/EML/CW、FAU、ABF/载板、T-glass、
             HVLP 铜箔、BT 树脂、探针/键合、MOCVD
Layer 4      变压器/并网、IPP/核电锻件、柴发、特气
```

|------|------|-------------|----------------------|----------------|

### 3.1 S 级清单（合并 7/2 + 7/4，本刊不新增未经核对的 S）

- InP 衬底 / EML（及逼近 S 的 CW）  
- ABF 膜 / 高阶载板；T-glass / Low-Dk 玻纤  
- HVLP4/5 铜箔；BT 树脂  
- 大型变压器 / 电力缺口；燃气轮机；数据中心柴发；核电大锻件  
- WF6、氦气等（纯正可交易标的稀缺或已利好出尽——见原图）  
- HBM underfill（Namics）等：**真瓶颈但不可投**

### 3.2 明确降级（必须写进 L0）

| 环节 | 7 月判定 | 关键证据类型 |
|------|----------|-------------|
| CoWoS | B：缺口 20%→10%，2027 转松 | 产能与缺口跟踪 |
| 液冷整机/冷板/CDU 整机叙事 | 非瓶颈；并购涌入 | 英维克利润下滑；CoolIT 等并购 |
| 光模块整机 | 非瓶颈 + 已透支 | 涨幅与毛利金丝雀信号 |
| 机器人「零部件瓶颈」 | 伪（需求数量级不够） | Optimus 量级 vs 扩产 |

## 5. 反向验证（摘要）

1. Capex 减速会先杀可取消订单的设备商；长约电力相对抗（亦见 7/2）。  
2. 材料扩产长，若 2027 需求降速，库存风险最大。  
3. Meta 出租闲置算力 ≠ 变压器交期变短，但削弱「电力叙事无条件做多」。  
4. 光子链 7/2 无利空暴跌 = 情绪，不等于 InP 逻辑坏；入场仍卡估值。
