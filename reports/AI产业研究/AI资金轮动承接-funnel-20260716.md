# AI资金轮动承接方向 — 漏斗式价值投资筛选

> **研究日期 / 数据截止：** 2026-07-16（CST）  
> **研究方法：** industry-funnel 四层漏斗（全市场扫描 → 5硬指标粗筛 → 精细分析 → 四大师深度）  
> **研究问题：** 上游芯片/存储/硬件大幅调整、资金流出后，美股 / A股 / 港股各自最值得重点关注的承接标的是什么？  
> **输出要求适配：** 常规漏斗终选 3 家；本报告按用户要求改为 **美股 3 + A股 3 + 港股 3**。  
> **一句话结论：** 资金并未系统性离开 AI，而是从「拥挤的上游铲子」转向「变现更清晰的平台/云」与「估值更低的中国科技」；三市终选为 **美股 GOOGL / META / APP，港股 腾讯 / 阿里 / 快手，A股 国电南瑞 / 同花顺 / 思源电气**。

本报告用于学习研究，不构成投资建议。

## 第三层：精细分析（终选 9 家）

报价时点以 **2026-07-15 前后** 为主；港美股来源 [stockanalysis](https://stockanalysis.com/)，A股年报来源巨潮/公司公告。

### 美股

#### Alphabet（GOOGL）— 核心

**一句话商业模式：** 卖搜索/YouTube广告给广告主，卖云计算给企业，自研模型与TPU降低单位推理成本。  
**财务质量：** 市值约 $4.3–4.5T；trailing PE ~27；ROE **35.8–38.9%**（stockanalysis ratios/statistics 两口径）；TTM OCF ~$174B、FCF ~$64B（Capex 高导致 FCF&lt;OCF）。  
**护城河：** 搜索分发+数据闭环+YouTube+Cloud backlog；★★★★★。  
**主要风险：** ①搜索被AI分流 ②Capex/FCF承压 ③监管。  
**估值快评：** 相对存储铲子不便宜，但相对自身质量与云增速 **合理偏贵**；承接「卖芯片买平台」最干净。  
**进入终选？** 是 — 核心仓。

#### Meta（META）— 成长

**一句话：** 社交注意力 → AI推荐 → 广告竞价变现。  
**财务：** 股价约 $676.64（7/15）；市值 $1.72T ✅（676.64×2.54B，偏差0.08%）；PE共识~24.3x；ROE~33%；OCF $124B / 净利 $70.6B ≈1.76；FCF $48.3B；D/E 0.36。  
**护城河：** 用户规模+广告主网络效应 ★★★★。  
**风险：** Reality Labs烧钱；广告周期；Capex上行。  
**估值：** Forward PE~20、PEG~1 → **合理**；近月相对领涨显示资金已在买变现端。  
**进入终选？** 是 — 成长仓。

#### AppLovin（APP）— 弹性

**一句话：** AXON 广告优化引擎向移动开发者卖效果广告。  
**财务：** 股价约 $452.44；市值 $152.0B ✅；PE~39 / fwd~26；PEG~0.70；净利率~64%；OCF≈FCF $4.44B；**D/E 1.63（黄）**；ROE被回购扭曲至极高。  
**护城河：** 算法与效果数据反馈 ★★★（弱于巨头）。  
**风险：** 高β、杠杆、广告监管、竞争。  
**估值：** 不便宜，但利润率与增速匹配度高于多数故事软件 → **贵但可作卫星**。  
**进入终选？** 是 — 弹性仓（仓位严格限制）。

> MSFT 与 GOOGL 同属核心，终选保留 GOOGL 以避免双仓重叠；MSFT 为高强度替换核心。

### 港股

#### 腾讯（0700）— 核心

**一句话：** 社交/游戏分发 + 广告 + 金融科技 + 云；AI嵌入广告与混元。  
**财务：** 股价约 474港元（7/15）；市值约 4.1–4.3万亿港元（股价×8.98B 与站点口径偏差约3.8%，⚠️可接受）；PE共识~16.2x；ROE 20.51%；TTM FCF约2620亿港元口径；D/E 0.33。1Q2026 收入1965亿人民币、经营现金流/FCF保持强劲（公司公告）。  
**护城河：** 微信生态转换成本+网络效应 ★★★★★。  
**风险：** AI收入占比披露不足；监管；南向波动。  
**估值：** 相对全球平台与自身历史 **合理偏便宜**。  
**进入终选？** 是。

#### 阿里巴巴（9988 / BABA）— 成长 ·黄

**一句话：** 电商现金流养云与大模型，抢中国AI基础设施与应用入口。  
**财务：** ADR市值约$271B，PE~17.6；港股市值 **stockanalysis 双源 1.72T（6月末）～2.01T（统计页）港元，偏差大，已标⚠️**，PE~15–17；**ROE 9.22%（不达标，标黄）**；D/E 0.25稳健；**FCF 显著为负**（云/即时零售Capex）。  
**护城河：** 电商规模+云客户迁移成本 ★★★★★。  
**风险：** ROE修复失败；Capex长期压制FCF；竞争加剧。  
**估值：** PEG线索偏低 → **不便宜于ROE，但便宜于云期权叙事**；必须标黄。  
**进入终选？** 是 — 成长仓（仓位低于腾讯）。

#### 快手（1024）— 弹性

**一句话：** 短视频+直播电商；可灵做生成式视频商业化。  
**财务：** 股价 44.70港元；市值 1935亿港元 ✅；PE 9.89x；ROE 23.74%；2025年利润186.24亿元人民币 ✅（年报/业绩公告双源）；OCF强但 **FCF近零/略负**（AI Capex）。可灵2025收入约10.4亿、ARR叙事抬升（研报/公司披露，需持续核实）。  
**护城河：** 下沉用户心智+电商闭环 ★★★。  
**风险：** 主业增速放缓；Capex抬升压利润；竞争。  
**估值：** 主业PE个位数 → **便宜**；AI弹性是期权。  
**进入终选？** 是。

### 4.1 美股终选

#### GOOGL（深写）

**段永平：** 这是「找信息/看视频的人」和「要算力的企业」两边收钱的生意；本分是产品与用户，别为AI叙事丢掉搜索体验。好生意，持续性来自习惯与分发。  
**巴菲特：**

| 护城河 | 强度 | 证据 |
|--------|------|------|
| 品牌/定价权 | ★★★★ | 搜索广告竞价 |
| 转换成本 | ★★★★ | Workspace/Cloud迁移成本 |
| 网络效应 | ★★★★★ | 广告主↔流量 |
| 规模效应 | ★★★★★ | TPU/数据中心 |
| 技术壁垒 | ★★★★ | 模型+芯片垂直整合 |

安全边际在「云增速与广告现金牛」对冲铲子周期波动，而非绝对低PE。  
**芒格：** 失败路径——①搜索份额结构性流失 ②Capex回报十年不够 ③反垄断拆分。聪明人可能不买：觉得AI Capex泡沫、FCF被抽干。  
**李录：** 属于文明级算力/信息范式；终局更可能是少数全栈平台赢家之一，而非纯应用。  
**综合：** 推荐度 ★★★★★｜核心｜建议：现价可建底仓，大涨后等回调加｜主题仓 **35–45%（美股篮子内）**｜监测：Cloud增速、FCF、Search体验指标。

#### META（压缩）
