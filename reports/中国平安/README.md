## 核心结论

中国平安是复杂但有护城河的综合金融机器：2025 年营运利润、NBV、财险 COR、分红均改善；当前估值与股息提供安全边际。关键约束是利率/利差损、银保周期与个险造血未恢复。稳健型可中长期布局，H 股安全边际通常优于 A。
