## 第一关：我能理解这门生意吗（能力圈）

| 必答问题 | 回答 | 判断 |

| 靠什么赚钱？ | M365/Azure/服务器与安全订阅或用量；Windows、LinkedIn、广告、游戏补充 | 说得清 |
| 10年后还在做什么？ | 大概率仍是企业生产力 + 云 + 安全 + 开发者平台，交互更多转向席位+Agent+算力 | 确定性较高 |
| 关键变量？ | Azure 增长与利用率、AI ARPU、Capex 回报、M365 留存、OpenAI、监管 | 可跟踪 |
| 认知来源？ | 年报、电话会、IR、结构化数据交叉 + 四角色团队报告 | 非道听途说 |

## 第三关：护城河够不够深（竞争优势）
