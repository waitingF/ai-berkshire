## 二、本期最重要的 3 个变化

1. **量起来了，利没跟上：收入大超预期，利润率与 EPS 大失预期**  
   交付 +25%、收入 +26% 明确结束“连续年度交付下滑”的阴影；但毛利率 16.8%（预期约 19.4%）、经营利润率仅 **1.4%**、Non-GAAP EPS **$0.33**（预期约 $0.50）说明：**量增被 ASP/结构、监管积分塌陷与费用膨胀对冲干净。** 这是典型的“顶线庆祝、底线受审”。

2. **费用与 CapEx 进入“期权建设期”，FCF 单季转负**  
   经营费用 +47% 至 $4.35B；CapEx +142% 至 $5.79B；FCF **-$1.1B**。公司明确：AI compute、太阳能、电池材料、半导体制造等多线产能在建；CFO 此前指引全年 CapEx **>$25B**，本季 run-rate 已指向高投入区间。汽车主业在给 Robotaxi / Optimus / 算力“输血”。

3. **利润表里的“一次性/金融项”开始扰动可读性**  
   Non-GAAP 调节显示：SpaceX 股权投资未实现收益（税后）约 **$763M**（每股约 +$0.22），另有某些税项 -$0.08。换言之，GAAP 净利 $1.11B 并不等于“造车赚钱 $1.11B”。同时监管积分收入从 $439M 降至 **$146M**，高利润附加项正在退去。

> 巴菲特要看的是可重复的所有者收益。本期可重复的部分在走弱：经营利润 -57%，FCF 转负；可讲故事的部分在变强：交付、FSD 订阅、Cybercab 投产、Optimus 产线。

## 四、附注中的隐藏信息

- [x] **监管积分塌陷**：Q2 仅 $146M（Q1 $380M，去年同期 $439M）。汽车毛利率“看起来”接近去年，是因为**剔积分后**毛利率从 15.0% 升至 16.3%；积分依赖下降是好事，但短期利润缺口真实存在。
- [x] **能源毛利率恶化**：约 **20.4%**（Q1 约 39.5%，去年同期约 30.3%）。电话会归因含约 **$240M** vendor cell 质保 true-up、Q1 关税利好不重复、工业储能 ASP 下行——可验证的质量/供应链风险，不是宏观空话。
- [x] **SpaceX 股权浮盈进入 Non-GAAP 调节**：税前未实现收益约 $1.005B，税后约 $763M。Tesla 与 SpaceX 的资产负债表联动增强，**主业利润可读性下降**。
- [x] **库存改善但应收略升**：库存天数 15（Q1 为 27）说明去库存成功；应收 $4.09B vs 收入大增，未见恶性塞货信号。
- [x] **SBC 大幅上升**：股票薪酬费用 $1.15B（去年同期 $0.64B），含 2025 CEO Performance Award 影响；稀释与费用是长期摩擦。
- [x] **FSD 订阅成为少数可见的软件锚**：Active FSD 订阅 1.48M（+56%）。仍非 Robotaxi 收入，但是“软件变现”少数可审计指标之一。

## 六、与投资论文的关系

既有研究（`特斯拉投资研究报告-20260702.md`）的核心判断是：

> **汽车+储能可验证盈利撑不起万亿市值；市场在为 FSD/Robotaxi/Optimus/AI 期权付费；安全边际不足则观望。**

本期验证：

| 论文支柱 | 影响 | 说明 |
|---------|------|------|
| 交付能否复苏 | **强化** | +25%，库存天数改善 |
| 汽车盈利能力 | **削弱** | 经营利润率 1.4%；积分塌陷 |
| 能源高质量增长 | **部分削弱** | 部署强，但毛利率受质保冲击 |
| 软件/FSD 变现 | **小幅强化** | 订阅 1.48M；仍非 Robotaxi |
| 期权叙事（Robotaxi/Optimus） | **事件强化、财务未强化** | 投产/扩城，但无收入拆分 |
| 估值依赖愿景 | **不变/更尖锐** | 利润 miss + 费用暴增，使“兑现期限”更敏感 |

## 七、结论：这份财报改变了什么？

1. **超预期 / 符合 / 低于预期？**  
   **收入与交付超预期，利润与自由现金流低于预期。**  
   综合判定：**运营复苏财报，不是盈利复苏财报。** 市场盘后下跌，与“量增利减 + 投资加码”的组合一致。

2. **对投资论文的影响**：**削弱“近期待汽车利润修复”路径，强化“公司主动牺牲近端利润换远期期权”路径。**  
   论文未破裂（交付与 FSD 订阅仍支撑长期叙事），但**近端安全边际进一步变薄**。

3. **下一个催化剂**  
   - **Cybercab / Semi 实际产量与收入确认时点**（投产 ≠ 贡献）  
   - **Optimus 首批下线后是否仍只用于内部训练**  
   - **能源质保（vendor cell）是否在后续季度继续侵蚀毛利**  
   - **2026Q3**：交付能否在高基数下维持；OpEx/CapEx 是否继续同比 +40% 以上  
   - **Robotaxi 单列 KPI**（出行次数、利用率、收入）——目前仍缺失

4. **若已持有：加仓 / 持有 / 减仓？**  
   **以持有/观望为主；不建议因交付超预期而加仓。**  
   - 加仓条件（观点）：经营利润率回到可持续的中高个位数，且 Robotaxi/软件出现可审计收入；或股价大幅下跌使汽车+储能现金流估值重新有安全边际  
   - 减仓条件（观点）：交付再次失速，同时 OpEx/CapEx 继续扩张导致净现金快速消耗；或能源质保/ASP 压力证明“量增”无法转化为单位经济性  

> 芒格：把“汽车公司”和“AI 期权”分开算。本期告诉你：期权还在烧钱，汽车主业刚把量拉回来，还没把利润拉回来。
