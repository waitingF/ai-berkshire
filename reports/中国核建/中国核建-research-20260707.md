**多方：**

**空方：**
- 2025归母净利润骤降36%，2026Q1继续下滑27%

- 工民建主动收缩但拖累整体营收和规模效应
- 子公司引入战投导致少数股东损益大增，归母利润被"分流"

## 第二步：生意本质分析 — 段永平"对的生意"

### 一句话定义

**中国核建 = 中国核电扩张的"施工国家队"，靠核岛建造的独占性资质和40年经验赚工程费，同时用核电品牌做低毛利工民建的横向扩张。**

### 商业模式画布

| 维度 | 分析 |
|------|------|
| 收入模式 | 工程建设（按履约进度确认收入），一次性项目制为主 |
| 客户 | 中核、中广核、国电投、华能等核电运营商；地方政府/央企（工民建） |
| 复购驱动 | 核电核准持续→运营商反复招标；核岛技术不可替代→高度锁定 |
| 硬件 vs 服务 | 纯工程建设（劳动+材料密集型），非平台、非订阅 |
| 定价权 | 核电工程强（资质壁垒），工民建弱（价格竞争） |

**段永平式追问：这门生意好在哪？**

好在核岛建造的**不可复制性**——全球唯一40年不间断造核电的企业，国内90%+核岛份额，新进入者即使有100亿也换不来40年经验、资质和核安全记录。但另一方面，工程承包本质上是"赚辛苦钱"：毛利率12%、ROE仅5%、需要持续融资垫资——这不是段永平最偏爱的"轻资产高回报"生意。

## 第三步：护城河评估 — 巴菲特"经济护城河"

| 护城河类型 | 强度 | 证据 | 趋势 |
|-----------|------|------|------|
| 品牌/定价权 | ★★★★（核电）/ ★（工民建） | 核岛工程毛利率13%+，工民建仅7-8% | 核电变宽，工民建变窄 |
| 转换成本 | ★★★★★ | 核电项目更换施工方风险极高（核安全监管） | 稳定 |
| 网络效应 | ★ | 无明显网络效应 | — |
| 规模效应 | ★★★★ | 同时承建40+台机组能力，全球规模最大 | 变宽 |
| 技术/专利壁垒 | ★★★★★ | 华龙一号/CAP1400/EPR/VVER等全堆型建造能力 | 稳定 |

**护城河综合评分：★★★★（4/5）——仅限核电工程板块**

工民建业务几乎无护城河（★1），但公司主动收缩该板块，整体护城河趋势**变宽**。

**巴菲特式追问：10年后这条护城河还在吗？**

核电建造的资质和经验壁垒在10年后大概率仍在——核电安全要求只会更严不会更松。但需警惕：如果SMR（小型模块化反应堆）技术成熟并改变建造模式（工厂预制替代现场施工），现有"现场施工经验"的价值可能被部分稀释。目前SMR尚处早期，10年内影响有限。

**芒格式追问：聪明人为什么不买？**

因为这是一门**ROE仅5%、需要持续股权融资（上市9年融资405亿）、应收账款为净利润33倍**的生意。核电景气度是"故事"，但财务报表是"现实"——2025年利润下滑36%说明故事尚未转化为股东回报。聪明的资金可能选择核电运营端（中国核电、中国广核）而非施工端。

## 第六步：行业与文明趋势 — 李录"文明演进框架"

### 文明级范式转移判断

**是——核电正处于新一轮投资周期**，但处于早期阶段：

| 维度 | 分析 |
|------|------|
| 范式转移 | 从"谨慎发展"到"积极有序发展"（原子能法2025年） |
| 技术革命类比 | 类似1970年代法国核电快速扩张期（美国因三里岛停滞，法国崛起） |
| TAM | 当前1.25亿千瓦→2035年1.8亿千瓦，增量约5500万千瓦，对应投资约1.1万亿 |
| 公司位置 | 产业链"卖铲人"——施工端，受投资周期驱动但无定价权 |
| 技术路线风险 | 华龙一号主导（安全），SMR/快堆/聚变尚早 |
| 客户集中度 | 极高——前四大运营商占94%市场份额 |

### 价值链位置

```
铀矿 → 燃料 → [核岛建造 ★核建] → 常规岛 → 运营 → 售电
                    ↑
              核建核心位置（高壁垒、中毛利）
```

**李录式追问：20年后回看，是"标准石油"还是"3Com"？**

更可能是**"铁路建设公司"而非"标准石油"**——在核电大扩张时代不可或缺、不可替代，但赚的是工程费而非资源/运营的超额利润。如果核电运营端（中国核电等）是"标准石油"，核建就是"修铁路的"——有存在价值，但股东回报取决于管理层能否在景气周期中实现利润转化，而非仅仅营收增长。
