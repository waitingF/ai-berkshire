## 第二步：生意本质分析 — 段永平"对的生意"

### 一句话定义

**中国中免是一门"政策特许+区位垄断+品牌集采"的免税零售生意——帮出入境/离岛旅客以低于有税渠道的价格购买进口奢侈品和消费品，赚取免税渠道与品牌之间的价差。**

### 商业模式画布

```
┌─────────────────────────────────────────────────────────┐
│  客户：出入境旅客、海南离岛旅客、即将离境的市内消费者      │
├─────────────────────────────────────────────────────────┤
│  价值主张：免关税/增值税/消费税，价格低于国内专柜10–30%   │
├─────────────────────────────────────────────────────────┤
│  渠道：离岛(53%) > 口岸机场(22%) > 市内/线上(25%)        │
├─────────────────────────────────────────────────────────┤
│  收入模式：商品销售（一次性）+ 少量租金/服务              │
├─────────────────────────────────────────────────────────┤
│  成本结构：商品采购(67%) + 租金/扣点 + 人工 + 物流       │
├─────────────────────────────────────────────────────────┤
│  关键资源：免税牌照、机场/海南区位、1600+品牌直采         │
└─────────────────────────────────────────────────────────┘
```

### 生意好在哪？

| 维度 | 评估 | 反面 |
|------|------|------|
| **复购驱动** | 弱——免税购物偏一次性/低频，依赖客流而非习惯 | 不像茅台/腾讯有强复购 |
| **定价权** | 中——免税价由政策决定上限，但品牌直采有5–10%成本优势 | 价格战（海旅、跨境电商）压缩空间 |
| **现金流** | 强——2025经营现金流60.6亿>净利35.9亿，净现金295亿 | 2022年曾出现负经营现金流（备货） |
| **经营杠杆** | 高——海南固定租金/折旧刚性，销售恢复→利润弹性大（2026Q1验证） | 下行周期杠杆反噬（2024–2025） |
| **可复制性** | 低——100亿可以开免税店，但拿不到核心机场标段和海南6店布局 | 牌照已扩容至10+主体 |

## 第三步：护城河评估 — 巴菲特"经济护城河"

| 护城河类型 | 验证 | 强度 | 趋势 |
|-----------|------|------|------|
| **品牌/定价权** | 1600+品牌直签，LVMH/Miller战略入股；但免税价受政策上限约束 | ★★★ | → 稳定 |
| **转换成本** | 低——消费者可在海南/机场/线上/跨境电商间比价 | ★★ | ↓ 变窄 |
| **网络效应** | 弱——5300万会员有数据价值，但非典型网络效应 | ★★ | → 稳定 |
| **规模效应** | 强——200店、8大保税仓、全球第二大免税运营商，采购成本领先5–10% | ★★★★ | → 稳定 |
| **牌照/区位** | 最强——海南6/12店、京沪核心机场、13城市内店；但牌照从垄断走向竞争 | ★★★★ | ↓ 边际弱化 |

### 护城河趋势判断

**过去5年**：2020–2021牌照垄断+海南爆发期护城河最宽；2022–2025竞争加剧（牌照扩容、出境分流、价格战）护城河边际收窄。

**未来5年预判**：
- 规模+供应链壁垒仍显著，短期难以被颠覆
- 牌照壁垒持续弱化（市内免税27家、机场招标引入外资）
- DFS/LVMH合作若成功，可能重建品牌护城河
- 海南份额仍在提升（2024+2ppt，2025H1+1ppt），但"一超多强"格局固化

**巴菲特式追问**：10年后这条护城河还在吗？**规模+区位护城河大概率还在，但牌照垄断护城河已不存在。** 什么能摧毁它？——政策取消免税、消费税改革、跨境电商完全放开。

### 空方核心论点（聪明人为什么不买）

1. **2021年峰值不可复制**：海南离岛免税601.7亿（2021）→304亿（2025），-49%。即使2026Q1反弹，全岛142.1亿（Q1）年化约570亿，仍低于峰值
2. **人次持续下滑**：2025离岛购物463万人次（-18.5%），"量跌价升"不可持续
3. **估值仍不便宜**：PE 29.4x对应6.87%净利率，盈利质量远逊于2021年（18.27%净利率）
4. **管理层零持股**：高管无股权激励，与股东利益绑定弱
5. **机场格局生变**：2026年浦东T1/S1归杜福睿，首都T2归王府井，中免从"独家"变为"之一"

**芒格式追问**：我最可能在哪里犯错？——**把2025Q4–2026Q1的政策脉冲反弹误判为可持续趋势反转。**

### 价值链位置

