## 核心结论（一句话）

> 厦钨是**钨钼深加工龙头 + 钴酸锂冠军 + 部分资源**的周期材料平台；2025 营收 462.65 亿 / 净利 23.09 亿真实，但存货暴增与 Q1 负 OCF 暴露周期库存本质。现价对应静态~32x（非东财~17x），中性情景无安全边际——**主题材料卫星可跟踪，加仓等存货与现金流改善。**
