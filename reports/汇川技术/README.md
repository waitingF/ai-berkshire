## 核心结论（一句话）

> 汇川是**中国驱动层工控龙头 + 新能源电驱第三方规模商**：工控侧份额与约 40% 毛利率真实可守，但近半收入来自约 16% 毛利的车电控，综合毛利约 29%、静态 PE≈32x——**好公司、不便宜；主题可作核心跟踪仓，宜等车业务毛利止跌或估值消化后再加仓。**
