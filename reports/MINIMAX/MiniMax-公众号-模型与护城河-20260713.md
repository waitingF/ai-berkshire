# 模型排第25，却最像一家公司：拆解 MiniMax 的护城河

在中国大模型"六小虎"里，MiniMax 的模型综合分不算高。最新的实时榜上，它的旗舰只排到全球第 25。

但它同时是这几家里最像一家产品公司、而不是纯模型实验室的一个：产品线最宽，出海变现最彻底，语音还是它唯一冲到过世界第一的模态。

这篇不聊解禁、融资和股价，那些是会过期的热点。这篇只回答两个能放很久的问题——它的模型到底排第几，护城河到底是什么。文中榜单分数、排名、定价都来自一手来源（Artificial Analysis 实时榜、官方文档、arXiv、港交所披露），凡是我自己的判断，会单独点明。

## 先认清现役阵容

很多人对 MiniMax 的第一印象还停在 M2。那是 2025 年 10 月那款点燃它存在感的编程与 Agent 爆款，开放权重、架构极稀疏、价格便宜，一度是开发者圈的热门。

但 M2 早已被一路迭代取代。截至 2026 年 7 月，家族现役主力是 5 月发布的 M3：原生多模态，支持文、图、视频输入，上下文 105 万 token，官方定价输入每百万 token 0.30 美元、输出 1.20 美元。

这个价格约为 Claude Sonnet 的 8%。换个说法：同样一批活，别人收一块，它收八分。便宜到这个程度，是理解它后面一切打法的起点。

所以谈"文本能力"时要分清：M2 是那款点燃声量的爆款，M3 才是现在扛榜的主力，两者不能混为一谈。

## 各模态到底什么水平

大模型公司很容易被笼统评价，但 MiniMax 各条产品线的水平差得很远。逐个模态看，才看得清它强在哪、弱在哪。

### 语音：唯一冲到过世界第一的模态

这是最亮的一块。在 Artificial Analysis 的语音合成竞技场（约 2026 年 7 月实时榜），它的 Speech 2.8 HD 拿到 Elo 1185、全球第 8，明显压过 ElevenLabs v3（1174，第 12）和 OpenAI 的 TTS。更早的 Speech-02-HD 在 2025 年上半年还曾登顶全球第一。

榜首是 Speechify 的 Simba 3.2，前列还有 Gemini、Cartesia 和阿里的产品。MiniMax 不在最顶端，但稳居第一梯队——这是它唯一冲到过世界第一、至今仍在第一梯队的模态，在六小虎里独一份。

### 文本、推理与 Agent：第二梯队，靠错位竞争

这是外界最关心、也最容易被高估的一块。

综合分上，M3 得 44 分、全球第 25。作为参照，头部是 Claude、GPT、Gemini 的 53 到 60 分档，国产开源的智谱 GLM-5.2 是 51。M3 落后头部约 10 到 16 分，也落后 GLM-5.2。这不是贬低，是事实：它就是第二梯队。（顺带打个假：网上有博客称"M2.7 以 50 分排第一"，与官方实时榜 M2.7 得 38 分、第 39 名严重矛盾，属杜撰，本文不采用。）

但拆到单项，会看到一个有意思的结构。

越是需要真本事从零推演的活，它越吃亏：SWE-bench、AIME、GPQA 这类硬推理硬编程，M3、M2 稳定落后头部数分到十几分。反过来，越是"调工具、拼产物"的活，它越能打平甚至反超：检索、前端产物、LiveCodeBench 这几项，它能追平乃至超过 Claude Sonnet。（其中检索项差距悬殊，可能与各家评测侧重不同有关，别单看一个数字下结论。）

这个结构说明一件事，这是我的判断：MiniMax 的文本能力不是靠"绝对更强"赢，是靠"够用，再便宜一个数量级"做错位竞争。活干得够好、价格只有头部的十分之一档，很多场景就够了。

但要清醒：这是价格差异化，不是能力差异化。而价格差异化最容易被跟进——对手一降价，或者干脆开源免费，这道差就抹平了。讲护城河时会回到这一点。

### 视频：掉队的一环，只剩中游加便宜

海螺（Hailuo）是它的视频线，曾有过高光——2025 年 6 月，Hailuo 02 一度冲到图生视频全球第二。

但一年之内被全面反超。现役旗舰 Hailuo 2.3 在 Artificial Analysis 视频竞技场里，文生视频前 25、图生视频前 12 都没进。榜前是字节 Seedance 2.0、快手可灵 Kling 3.0、Google Veo 3.1、阿里 Wan 2.7 这些名字。

官方现在主打两点：物理表现和成本——每条约 0.28 到 0.30 美元，对比 Runway 约 1.20、Sora 约 3.50。但它没有原生音轨，榜单也确实掉了队。视频是个三个月换一次王座的红海，海螺在其中只剩"中游加便宜"这一个卖点。

三个模态摆一起，画像就清楚了：语音最强、文本靠低价错位、视频落后头部。这不是一个靠单点最强取胜的选手，是一个走性价比和开放权重路线的第二梯队全能选手。

## OpenRouter 上的真实分量：一个反差

榜单是评测环境，OpenRouter 才是开发者用真金白银投票的地方，最能反映真实调用量。

先说一句必要的提醒：OpenRouter 的用量和花费数字都是二手汇总、来源互相冲突，只能看个量级，不能当硬数据。真正能硬引的只有官方定价。

在这个前提下，量级大致是：M3 的月 token 吞吐约 4.5 万亿，在编程类里约排第二，MiniMax 家族合计约占全站 3.8%。作为参照，按 token 榜首是 DeepSeek，单家约占全站 17.6%。MiniMax 不是那种统治级第一，也不是长尾，是靠编程和 Agent 场景加低价撑起的次头部。

真正值得讲透的是这个反差：按 token 用量，它是准头部；按美元花费，它只是中游。

原因就在它的打法本身——量大、价低、毛利薄。对照头部：全站按美元花费第一的是 Anthropic，占了约六成七，靠的是量少价高。MiniMax 恰好相反。这个"token 份额很高、却换不成同等比例收入"的结构，既是它的打法，也是它的软肋。用量榜上好看，不等于收入榜上好看。

## 护城河逐条盘点

现在回到第二个问题：MiniMax 的护城河到底是什么。

先给我的结论：它没有一条又宽又深的河。它的壁垒是多个中等深度差异点的叠加，外加早期形成的出海消费数据和分发飞轮。逐条看证据强弱，正反两面都摆出来。

**成本与长上下文经济学，★★★★。** 证据最硬的一条。M1 的全流程强化学习训练只用了 512 张 H800、约三周、成本约 53 万美元；支持百万级上下文，算力开销约为对手的三成；M2 价格做到 Claude 的 8%。反面：架构优势可以被复制，DeepSeek、Qwen 也在做稀疏和线性注意力，这条河会被跟进。

**出海收入占比超过 70%，★★★★。** 六小虎里最彻底的一家，是真差异化。反面：高出海占比把公司暴露在地缘和合规风险下，海外陪伴产品 Talkie 就曾在美区应用商店被下架。

**多模态全栈广度，★★★。** 文本、视频、语音、音乐、Agent、C 端陪伴 App 一家做齐，产品线是这几家创业公司里最宽的。反面：宽不等于深，每条线都有更专的对手，视频掉队就是证据。

**AI 陪伴 App 的数据和分发，★★★。** Talkie 曾有过半用户在美国，有真实收入和用户资产，这是纯 API 型对手不具备的。反面：留存下滑加监管下架风险，陪伴类的忠诚度和合规风险双高。

**Agent 与编程的极致性价比，★★。** OpenRouter 编程类前二是硬数据。反面：低价本身就容易被商品化，DeepSeek、GLM 更便宜、甚至免费开源。

**开放权重的开发者生态，★★。** 有一定号召力。反面：开源不是它独占的打法，DeepSeek、Qwen、GLM 更彻底，它在这条上并不占先。

把这六条连起来，有一个值得警惕的隐忧，也是我认为全篇最关键的一句：证据最强的两条是成本经济学和出海占比，最弱、最容易被侵蚀的两条是开放权重和 Agent 低价——而恰恰是后两条，撑起了它当下在 OpenRouter 上的声量。让它今天显得热闹的那部分，护城河反而最浅。

## 四条反面风险

护城河讲能守住什么，风险讲可能被什么击穿。有四条要摆在桌面上。

一是财务还没自我造血。2025 全年收入 7900 万美元，同期经调整净亏损约 2.5 亿美元，收入高速增长但离盈亏平衡还远。好在账上有约 10 亿美元现金，加上今年的再融资，按一年烧 2.5 亿算，跑道还有几年——不紧迫，但也谈不上宽裕。

二是开源等于零迁移成本，会催生持续的价格战通缩。客户换供应商几乎没成本，DeepSeek、GLM 的免费开源会一直往下压它的 API 变现空间。

三是投资人本身就是对手。阿里既是它重要的机构股东，又用 Qwen 正面竞争；字节和腾讯的资本加分发是碾压级投入。这是创业公司在巨头夹缝里做全栈的结构性困境。

四是消费 App 的监管与地缘风险。Talkie 美区下架、陪伴类的隐私审查，都会因为它超过 70% 的高出海占比而被放大。

## 和六小虎其他家比，它赢在哪

把它和 DeepSeek、Kimi、Qwen、GLM 放一起：论绝对模型分，它落后 GLM-5.2 和 DeepSeek；论 OpenRouter 的 token 统治力，它 3.8% 远不及 DeepSeek 的 17.6%。单看这两个维度，它都不是最强的。

但换个维度就不一样，这是我的判断：论"产品公司气质"，MiniMax 是这几家里最独特的一个。多模态全栈做齐、出海消费 App 有真实付费用户、已经跑通商业化闭环——这三件同时做到规模的，只有它。DeepSeek 长在纯推理和开源影响力，Kimi 长在 Agent 和编程，Qwen 长在阿里云生态，GLM 走企业加开源。产品形态的完整度上，MiniMax 最像一家公司、而不是一个实验室。

所以它赢的不是模型分，是商业形态的完整度。用模型排行榜衡量它，会低估它；用它的模型排名追捧它，又会高估它。

## 结尾：一场和时间赛跑的生意

把两个答案收拢。

模型水平：第二梯队。语音第一梯队，文本靠低价错位、够用但不领先，视频掉队。护城河：不是一条深堑，是几道中等深度差异点的叠加，加上一个早期跑起来的出海消费飞轮；最强的是成本效率和出海占比，最容易被填平的是开放权重和低价——而后者正是它当下声量的来源。

所以对 MiniMax 最经得起推敲的判断是：它做的是一门和时间赛跑的生意。它赌的不是模型永远领先，因为在一个技术每隔几个月就换一次王座的行业里，模型领先本就留不住。它真正赌的，是能不能在自己的低价和开放权重被彻底商品化之前，先用多模态的宽度和出海的分发，把用户的使用习惯锁住。

值得盯的就两个变量：出海的消费飞轮能不能在壁垒被填平之前转到自我造血，以及高出海占比下的合规会不会失控。前者去看它的 ARR 增速能否持续跑赢烧钱，后者去看新规落地后海外那块营收掉多少。

在一个王座频繁易主的行业里，唯一不太贬值的资产，是客户改不掉的习惯。MiniMax 能不能守住它，答案不在模型榜上，在时间里。
