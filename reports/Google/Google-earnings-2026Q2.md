## 二、本期最重要的 3 个变化

1. **Cloud 从“加速”变成“断层式加速”，利润率同步跳升**  
   Cloud 收入 $24.8B、**+82%**（Q1 为 +63%），经营利润 $8.8B、利润率约 **36%**（去年同期约 21%）。这是本期真正改变叙事斜率的数字：AI 基础设施与 AI solutions 已在收入与利润两端同时体现。但另一方面，CapEx 单季已达 $44.9B，几乎是 Cloud 单季收入的 1.8 倍——增长质量高，资本强度也高。

2. **账面利润被股权浮盈“吹胀”，经营现金却撑不住 CapEx**  
   GAAP 净利 $112.1B、EPS $9.11，其中股权证券净收益约 **$99.0B**，对稀释 EPS 贡献约 **$6.26**。剔除此项后核心 EPS 约 **$2.85**，略低于共识。更关键的是：经营现金流 $39.1B，CapEx $44.9B，**单季 FCF -$5.9B**——Alphabet 进入“经营利润很美、自由现金暂时让位于 AI 军备”的阶段。

3. **融资结构切换：停回购 + 股权/债权双募资，为 AI 产能买单**  
   Q2 回购为 **$0**（去年同期回购 $13.2B）；6 月发行普通股+强制转优先股净募约 **$49.6B**，另发债净募约 **$20.3B**；并设立最高 **$40B** ATM 额度（季末尚未动用，主要用于员工股权税务）。这不是“现金流紧缺到生存危机”，而是管理层主动把资产负债表改造成 **AI 产能弹药库**。对股东而言：稀释与利息成本上升，换取供给约束下的份额与 backlog 兑现。

> 巴菲特看所有者收益，不看浮盈。本期要盯的不是 $9.11，而是 Cloud ROIC 与 FCF 何时重回正增长。

## 四、附注中的隐藏信息

- [x] **股权浮盈主导账面净利**：股权证券净收益 $99.0B；附注明确其使税项、净利、稀释 EPS 分别增加约 $21.9B / $77.1B / $6.26。**任何用 GAAP PE 看 Alphabet 的估值都会严重失真。**
- [x] **存货暴增**：存货从 2025 年末 $2.44B → $9.99B（约 **+310%**）。结合 Cloud 含 TPU 系统销售、数据中心建设，更像 **AI 硬件/设备备货**，但仍需在 10-Q 附注确认构成，警惕渠道或硬件确认节奏扰动。
- [x] **商誉跳升**：商誉 $33.4B → **$57.8B**（约 +73%）。与 YTD 收购（含 Intersect 等）一致；后续减值与整合执行是新风险点。
- [x] **应收账款**：应收 $62.9B → $69.2B（约 +10%），慢于收入 YoY +24%，**未见明显“放松信用冲收入”信号**（半年度对比需结合季节性）。
- [x] **回购停、股权激励现金税负上升**：为满足员工股权税务，设 $40B ATM；Q2 与股权奖励相关净支付 **$6.6B**（去年同期 $2.6B）。
- [x] **Alphabet-level AI 成本上升**：未分配成本 -$5.8B（去年 -$3.4B），说明通用大模型研发仍大量在集团层，**分部利润率可能高估“完整 AI 成本”**。
- [ ] **客户集中度 / 关联交易细节**：本期 earnings release 未披露；待 10-Q。

## 六、与投资论文的关系

仓库内既有研究（`Google-Alphabet投资研究报告-20260623.md`、`Google-checklist-20260718.md`）的核心张力是：

> **广告+云护城河仍在，但生意正在变重；安全边际取决于 CapEx 回报，而非叙事热度。**

本期财报对论文的冲击：

| 论文支柱 | 影响 | 说明 |
|---------|------|------|
| Search/广告仍是现金牛 | **强化** | +17%，利润率仍高；但 Q3 高基数提示增速或放缓 |
| Cloud/AI 是第二增长曲线 | **显著强化** | +82% + 利润率跳升，是迄今最强证据 |

## 七、结论：这份财报改变了什么？

1. **超预期 / 符合 / 低于预期？**  
   **经营层面超预期，资本市场叙事低于预期。**  
   - 收入与 Cloud：**超预期**  
   - 核心 EPS：**略低于预期**  
   - CapEx 指引：**高于市场舒适区**（上修）  
   综合判定：**经营强、资本开支更强——对“利润表买家”偏多，对“自由现金流买家”偏空。**

2. **对投资论文的影响**：**强化 Cloud/AI 兑现，同时削弱“轻资产高 FCF”旧画像**。论文未破裂，但权重从“广告现金牛折现”进一步滑向“AI 基础设施 ROIC 能否证明”。

3. **下一个催化剂**  
   - **2026Q3**：Search 高基数下的真实增速；第三方算力对 Cloud/集团利润率的拖累幅度  
   - **CapEx 执行**：下半年还需约 **$120B** 量级（以指引中枢 $200B - H1 $80.6B 粗算）能否转化为 Cloud backlog 收入  
   - **10-Q 附注**：存货构成、商誉新增明细、股权投资公允价值敏感性

4. **若已持有：加仓 / 持有 / 减仓？**  
   **持有为主；不宜因 GAAP 暴利加仓，也不宜因单季负 FCF 恐慌性减仓。**  
   - 加仓条件（观点）：核心 EPS 与 Cloud 利润率在 CapEx 高压下仍能扩张，且 FCF 在未来 2-3 季回到可持续正值  
   - 减仓条件（观点）：Cloud 增速快速回落、第三方算力成本变成结构性利润率侵蚀，或股权融资/ATM 启动后稀释失控  

> 芒格：反过来想。如果 Cloud +82% 都买不进市场，真正的否决票写在 CapEx 上——市场在问：这些千瓦和机柜，最终会变成所有者现金，还是变成永续军备竞赛？
