## 二、竞争格局总览

### 2.1 战略地图（2026中）

| 赛道 | Alphabet位置 | 主要对手 | 竞争本质 |
|------|--------------|----------|----------|
| Search / 信息入口 | 传统搜索仍绝对领先 | Bing/OpenAI/Apple/Perplexity | 习惯+默认分发+答案体验 |
| 数字广告 | 仍巨头，份额承压 | Meta、Amazon | 注意力与闭环转化 |
| Cloud | 清晰第三，增速第一梯队 | AWS、Azure；neocloud | AI基础设施+企业关系 |
| 视频广告 | YouTube大体量，增速减速 | TikTok、Netflix Ads、Meta Reels | 时长×履约×下漏斗 |
| 移动OS | Android全球约七成 | iOS | 入口控制权（被监管重塑） |
| 前沿模型/助手 | 全栈追赶者兼整合者 | OpenAI、Anthropic、Meta、Microsoft | 模型质量×分发×云捆绑 |

### 2.2 竞争策略对比（精简）

| 对手 | 核心策略 | 对Alphabet的含义 |
|------|----------|------------------|
| **Microsoft + OpenAI** | 把助手嵌进Windows/Office/GitHub/Azure；用分发换模型领先 | 企业「默认AI」争夺；Bing份额仍小但桌面/办公有楔子 |
| **OpenAI（独立）** | ChatGPT成为新一代「提问习惯」；搜索/购物能力扩展 | 分流**意图形成前**的对话流量；尚未等价替换商业搜索 |
| **Apple** | 控制iOS默认；可谈判搜索分成与AI助手入口 | 不是搜索引擎，却是最贵的「分发阀门」 |
| **Meta** | Advantage+自动化 + 全漏斗社媒广告 + 新广告面（WhatsApp/Threads等） | 广告预算再分配的最大受益者候选 |
| **Amazon** | 零售媒体闭环（购买意向数据） | 侵蚀高意图效果广告预算，尤其零售/CPG |
| **AWS** | 规模、企业关系、自研芯片、生态深度 | Cloud利润池守门人 |
| **Azure** | 企业身份/Office绑定 + OpenAI独家分发优势 | Google Cloud最大「关系型」对手 |
| **Anthropic** | 企业API/编码场景强势 | 抢GCP上的模型工作负载与「智能层」议价权 |
| **Meta Llama** | 开源权重压低推理成本 | 长期压模型层毛利，利好有算力与分发者 |

> **芒格点评（能力圈）**：Alphabet的能力圈是「组织全世界信息 + 大规模广告拍卖 + 数据中心工程」。它不是天生的「社交关系图谱公司」，也不是「零售成交闭环公司」。在自己能力圈里打仗（搜索、YouTube、云AI）胜率高；跑去和Meta拼信息流创意广告、和Amazon拼货架广告，是在别人的主场。

## 三、核心竞争者威胁评估

评分说明：威胁★ = 对Alphabet中期（3–5年）自由现金流与护城河的潜在破坏力（5=极高）。

### 3.1 逐个评估

| 竞争者 | 威胁★ | 证据强度 | 关键判断 |
|--------|-------|----------|----------|
| **Meta** | ★★★★☆ | 高 | 广告收入增速显著更高；eMarketer预测2026反超——**广告侧最大现实威胁** |
| **OpenAI / ChatGPT** | ★★★★☆ | 中高 | 习惯层分流强；收入层尚未取代Search；零点击与「答案即终点」改变搜索产品形态 |
| **Amazon Ads + AWS** | ★★★★☆ | 高 | 广告端零售媒体；云端仍是份额王+高利润；双线施压 |
| **Microsoft Azure + Copilot** | ★★★★☆ | 高 | 企业默认入口；云份额第二；OpenAI分发放大器 |
| **Apple（分发/默认）** | ★★★★☆ | 高 | 不靠产品打败Google，靠**合同与系统默认**抽血或改规则 |
| **Anthropic** | ★★★☆☆ | 中 | 企业API/编码领先叙事强；对Cloud模型层有分流，对Search广告威胁间接 |
| **TikTok** | ★★★☆☆ | 中高 | 抢YouTube增长与年轻用户发现；WARC指YouTube广告增速放缓 |
| **Perplexity等AI搜索** | ★★☆☆☆ | 中 | 叙事热、体量仍小；监管与版权摩擦多 |
| **Neocloud（CoreWeave等）** | ★★☆☆☆ | 中 | AI算力细分抢单；总云市场约5%，但改变定价与交付预期 |

### 3.2 深度点评（非共识优先）

#### （1）Meta——广告双寡头是否松动？
- **事实**：eMarketer预测Meta 2026全球数字广告净收入$243.5B vs Google $239.5B；三强合计约62%份额（多家转述eMarketer）。
- **但另一方面**：Google Search仍是高意图效果广告主阵地；YouTube在CTV/长视频仍有独特库存。
- **观点**：格局不是「崩塌」，而是从「Google略强的双寡头」走向「Meta-Google胶着 + Amazon第三极」的**三强再平衡**。对Alphabet最痛的是**相对增速与广告主钱包份额**，不是绝对收入归零。

#### （2）OpenAI / AI助手——搜索是否被分流？
- **事实**：
  - StatCounter：2026年6月全球传统搜索份额 Google **91.27%** / Bing **4.68%**。
  - Alphabet Q1'26：Search & other **$60.4B，+19%**；管理层称query创新高，AI体验驱动回访。
  - SparkToro/Similarweb相关：2026年前四月美国Google搜索约**68%零点击**；AI Overviews出现时CTR显著下降。
  - 出版商侧：Chartbeat等显示Google引荐大降（小站约-60%、中站-47%、大站-22%量级），而ChatGPT等AI引荐仍**<1%**页面流量。
- **非共识**：「Google被ChatGPT打死」与财报**同时为假**——至少目前如此。更准确的说法是：
  1. **注意力与提问习惯**正在被助手分流；
  2. Google用AI Overviews/AI Mode把答案留在自有界面，**保住查询与广告**，同时**削弱开放网页**；
  3. 真正风险是长期：**商业查询被对话式购物/代理完成**，或监管迫使默认与数据开放，使「答案墙」策略成本上升。

> **逆向思考**：杀死搜索现金牛的，不一定是「另一个搜索框」，而可能是「用户不再需要点击任何广告」+「广告主发现闭环在Amazon/Meta更划算」+「默认分发被法院拆掉」。

#### （3）AWS / Azure——Cloud真实位置
- **事实**：份额 28% / 21% / 14%；收入体量 AWS ≈ 1.9× Google Cloud（Q1）；但GCP **+63%** 显著快于AWS/Azure约+28%档；Cloud backlog披露超 **$460B**（环比近翻倍，公司口径）。
- **观点**：Google不是「永远老三的失败者」，而是「用AI需求换档追赶的第三名」。但份额每升1个点都极难；企业多云与切换成本意味着**追赶是马拉松不是百米**。
- **反过来想**：若AI capex回报不及预期，最先受伤的是高增长叙事下的GCP估值溢价，而非AWS的存量利润。

#### （4）Apple——被忽视的「非搜索竞争对手」
- 不生产搜索引擎，却掌握Safari/Siri/Spotlight默认。
- DOJ救济阶段：排他性默认受限，但**非排他分成安排仍可能存在**（以法院最终生效文本与上诉结果为准）。
- **观点**：Apple对Google的威胁，常被低估为「TAC成本」；真正风险是**默认权货币化方式改变**或**系统级AI助手绕过Google**。

#### （5）监管——改变竞争规则的「第三只手」
- 美：Mehta救济偏行为性（禁排他默认、数据共享/联合销售等），拒绝强制剥离Chrome；2026年政府与多州**交叉上诉**寻求更强救济；Google亦上诉——**终局未定**（Reuters；Goodwin 1H2026综述）。
- 欧：DMA规格程序要求向第三方AI助手平等开放Android相关能力、并以FRAND分享匿名搜索数据等（Goodwin转述欧委会2026进展；最终决定以官方文本为准）。
- **观点**：监管很少「立刻杀死」利润，但会提高**对手获得冷启动数据与分发**的概率——这是典型芒格所说的「改变激励机制」。

## 六、产业链价值分配（芯片 → 云 → 模型 → 应用 → 广告）

```text
[上游] 半导体/服务器/电力/土地
   ↓  利润：极厚（稀缺产能期）
[中上游] 云IaaS/PaaS/加速器租赁（AWS/Azure/GCP/neocloud）
   ↓  利润：厚，且AI扩容期capex巨大
[中游] 基础模型与API（OpenAI/Anthropic/Google/Meta开源）
   ↓  利润：分化剧烈；开源压价；头部API可观但不稳定
[中下游] 应用/助手/Agent/办公套件
   ↓  利润：看分发与工作流嵌入
[下游] 广告拍卖与零售媒体（Google/Meta/Amazon）
      利润：传统上最厚、最稳 —— 正在被重新分割
```

| 层级 | 价值捕获特征（2026） | Alphabet卡位 | ★（价值捕获力） |
|------|----------------------|--------------|-----------------|
| 芯片/系统 | Nvidia等仍强；自研ASIC分化 | TPU自研降低单位成本 | ★★★★☆ |
| 云 | AI需求推高增速与议价 | GCP第三但加速 | ★★★☆☆→★★★★☆ |
| 模型 | 企业API份额争夺白热化 | Gemini全栈，但习惯落后ChatGPT | ★★★☆☆ |
| 应用/分发 | 入口决定一切 | Search/YouTube/Android/Workspace | ★★★★★ |
| 广告变现 | 高意图仍贵；社媒/零售分流 | Search+YT强，社媒弱，零售弱 | ★★★★☆ |

**产业链结论（观点）**：Alphabet少有地同时坐在**中上游（云/芯片）+ 下游（广告）**。这是优势也是冲突——下游广告要靠「把用户留在答案里」，中游模型要靠「消耗更多token」，上游云要靠「客户把负载放在GCP」。全栈只有在各层目标一致时才是飞轮；目标冲突时就是「自己人打自己人」。

## 七、芒格清单：什么会杀死Google？显而易见却被忽视的风险

### 7.1 逆向：杀死Google的候选剧本

| 剧本 | 概率（观点） | 杀伤力 | 当前观测 |
|------|--------------|--------|----------|
| A. ChatGPT直接夺走搜索广告收入 | 低–中 | 极高 | 尚未发生；Search仍+19% |
| B. Meta+Amazon持续夺走广告增量 | 中–高 | 高 | eMarketer份额预测已体现 |
| C. 默认分发/数据共享救济实质生效 | 中 | 高 | 上诉期，落地节奏不确定 |
| D. AI答案导致广告ROI长期恶化 | 中 | 高 | 零点击↑；商业影响滞后于流量影响 |
| E. Cloud AI投资回报不及预期拖累利润 | 中 | 中高 | Capex高企是行业共性 |
| F. 开源+neocloud压垮模型/算力溢价 | 中 | 中 | 已发生在边缘场景 |
| **G. A×B×C×D 叠加（Lollapalooza）** | 低但非零 | **致命** | 这是真正要盯的尾部 |

### 7.2 「显而易见却被忽视」

1. **出版商生态坏死**：Google可以在零点击下继续赚钱，但开放网页萎缩会反噬长期搜索质量与监管合法性——慢性病，不是季报病。  
2. **统计幻觉**：91%搜索份额与「有效信息获取份额」不是同一个东西；用错分母会得出错误安全感。  
3. **Apple不是朋友**：TAC是成本，也是共生；共生破裂比多一个Bing更可怕。  
4. **广告三强不是「双寡头微调」**：Retail media是结构性第三极。  
5. **Cloud胜利不等于Search永续**：GCP+63%不能对冲Search产品形态被改写的长期风险。

> **芒格**：避免蠢事的第一条——别把「还没在财报上出现」当成「不存在」。

## 九、总体结论（事实 / 观点；正反两面）

### 事实（有公开来源支撑）
1. 数字广告仍是万亿级媒介结构的主引擎；Google、Meta、Amazon高度集中（约六成量级）。  
2. Alphabet Q1'26：总收入$109.9B（+22%）；Search & other $60.4B（+19%）；YouTube ads $9.9B（+11%）；Cloud $20.0B（+63%），backlog>$460B。  
3. 传统搜索份额：Google约91%（StatCounter，2026-06）。  
4. 云基础设施：AWS/Microsoft/Google约28%/21%/14%（Synergy，Q1'26）；市场约+35% YoY。  
5. AI总支出预测约$2.59T（Gartner，2026）；基建是最大支出块。  
6. 搜索零点击与出版商引荐下降有多源观测；AI引荐占比仍极低。  
7. 美国救济与上诉、欧盟DMA规格程序仍在进行，终局未完全落地。

### 观点（明确标注）
1. **搜索「被AI分流」被市场讲成了生死题，财报阶段更像「产品形态题」**：流量契约在变，广告现金牛尚未倒。  
2. **广告双寡头确在松动**——不是Google突然变弱，而是Meta增速与Amazon零售媒体把增量切走；2026若Meta年收入反超，将是标志性心理拐点。  
3. **Cloud的真实位置是「加速的第三」**：份额故事比两年前好看，仍不是AWS替代者。  
4. **最大被忽视风险不是Bing，而是Lollapalooza**：习惯转移 × 广告再分配 × 默认/数据监管 × AI变现效率。  
5. 在芒格框架下，Alphabet仍处「好生意被新物理冲击」的阶段——**能力圈宽、资产负债表与分发强，但投资者最容易犯的蠢事是用旧护城河公式外推十年。**
