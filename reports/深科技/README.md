## 核心结论（摘要）

深科技是「EMS 底盘 + 存储封测成长腿」的混合公司，而非纯长鑫标的。长鑫招股书具名向合肥沛顿采购封测（S级），贴厂逻辑成立；但委外份额 60%–70% **未在公司财报写死**，与 40%/三分天下等口径冲突。2025 营收 157.47 亿、归母净利 11.36 亿、ROE 9.09%、OCF/净利约 155%；2026-07-20 市值约 553 亿、PE(动)~46。同业相对不贵，绝对安全边际不足；中性三情景目标价≈现价。

**操作态度：主题可跟踪，不宜追高；优先等待份额书面澄清或估值进一步回落后再加仓。**
