## 摘要：核心结论先行

| 维度 | 结论 |
|------|------|
| 主题定义 | 先进制造（自动化/机床/机器人/传感）+ 先进材料（碳纤维/关键金属/电子材料）双轮；**不是**单一申万行业 |
| 行业位置 | **扩张期中段、结构分化**：宏观产量验证强；微观上质量股贵、故事股更贵、部分周期材料相对克制 |
| 终选 3 家 | **汇川技术（核心）/ Keyence（质量卫星）/ 厦门钨业（材料卫星）** |
| 组合逻辑 | 中国工控成长兑现 × 全球传感现金牛 × 钨钼深加工利润弹性 |
| 最大反直觉发现 | **卡脖子最紧的标的多数进不了终选**（绿的谐波、Nittobo、Almonty、铂力特）——稀缺已计价 |
| 最大风险 | 制造业 CAPEX 下行；汇川新能源稀释工控纯度；Keyence 估值永久溢价；厦钨存货与商品价格 |
| 主题仓位上限（观点） | 总仓位 **8–15%** |

> 芒格：「反过来想。」——当所有人都在买「最卡脖子」时，先问：价格是否已经卡死了回报？

## 第三步：精细分析（保留公司）

### 汇川技术（300124）

**一句话商业模式：** 卖工业自动化产品（伺服/变频/PLC 等）与新能源车动力域电控，向制造业客户与车企收费。

**财务质量：** 2025 营收 451.05 亿（+21.8%），归母净利 50.50 亿（+17.8%），扣非 +22.7%；毛利率 28.95%；ROE 16.34%；OCF 66.81 亿 / 净利 1.32×。关键变化：工控与车电控双轮；人形机器人仍是早期期权。

**护城河：** 国内渠道+产品组合+切换成本；5 年后大概率仍在，但全球对标毛利差距仍大。

**主要风险：** ①工控周期；②车电控价格战；③机器人预期管理失败。

**估值快评：** TTM PE 32×，动态约 40× → **合理偏贵**。

**进入终选？** **是** — 硬指标全过，主题核心。

### Keyence（6861.T）

**一句话：** 直销高毛利工业传感器与机器视觉，卖「检测与控制前端」。

**财务质量：** TTM 收入约 ¥1.17T，净利约 ¥445B；毛利率约 83%；净现金；FCF 约 ¥402B；ROE 13.5%。

**护城河：** 产品库+直销组织+客户嵌入，五星级。

**主要风险：** ①估值；②全球工厂 CAPEX；③日元。

**估值快评：** PE 38.6× / PS 14.7× → **贵，但是好公司**。

**进入终选？** **是** — 作为质量锚（巴菲特型）。

### SMC（6273.T）

**一句话：** 全球气动元件标准件龙头，工业「水电耗材」。

**财务质量：** PE≈26×，毛利率约 45%，ROE 仅 8.3%，扩产压制 FCF。

**进入终选？** **否（可替换 Keyence）** — 估值更友好，但资本回报弱于 Keyence；终选保留更高生意质量。

### 厦门钨业（600549）

**一句话：** 钨钼深加工 + 稀土 + 能源新材料平台。

**财务质量：** 2025 营收 +30.8% 至 462.65 亿，净利 +34.9% 至 23.09 亿；OCF/净利 1.29×；ROE~14%；负债率~52%。黄灯：存货 +73%。

**护城河：** 加工认证+资源协同；商品价格是双刃剑。

**主要风险：** ①钨价；②存货跌价；③出口管制。

**估值快评：** TTM PE 32.4×，动态更低 → **主题材料股中相对合理**。

**进入终选？** **是** — 材料侧能过筛的稀缺标的。

## 第四步：四大师深度分析（终选 3 家）

### 4.1 汇川技术 — 核心

#### 段永平：生意本质
做中国制造业的「动力与控制基础设施」。本分是把产品做好、渠道做深；人形机器人是期权不是本分。持续性来自制造业自动化渗透与国产替代份额。

#### 巴菲特：护城河

| 护城河 | 强度 | 证据 |
|--------|------|------|
| 品牌/定价权 | ★★★ | 份额升，价格战仍在 |
| 转换成本 | ★★★★ | 产线控制系统 |
| 网络效应 | ★★ | 生态建设中 |
| 规模效应 | ★★★★ | 研发与渠道 |
| 技术壁垒 | ★★★ | 电力电子积累 |

安全边际：等待 TTM PE 回至 ≤28× 或工控订单加速时加仓。

#### 芒格：失败模式
1. 工控长期价格战吞噬 ROE  
2. 新能源车电控客户集中与降价  
3. 管理层过度押注机器人故事  

最坏情景：利润率下台阶，市值向「优质周期股」中枢回归（粗估较现价 -30% 量级，**估计**）。

#### 李录：文明定位
中国先进制造能力提升是文明级赛道上的国家工程；汇川是其中可交易的平台型工具，而非热潮玩具。

#### 综合推荐
```
推荐度：★★★★★
仓位类型：核心
建议买入区间：现价可建底仓；更优买点=回调 10–15% 或 TTM PE≤28×
建议持仓比例：占该主题仓位 50–55%
关键监测：工控收入增速、毛利率、新能源业务利润贡献、机器人订单是否「故事化」
```

### 4.2 Keyence — 质量卫星

#### 段永平
好生意教科书：高毛利、直销、净现金、少资本开支。本分清晰，不造整机机器人却赚自动化最肥的一层。

#### 巴菲特：护城河

| 护城河 | 强度 | 证据 |
|--------|------|------|
| 品牌/定价权 | ★★★★★ | 毛利率~83% |
| 转换成本 | ★★★★ | 嵌入产线检测 |
| 网络效应 | ★★★★ | 产品库交叉销售 |
| 规模效应 | ★★★★ | 全球直销 |
| 技术壁垒 | ★★★★ | 持续新品 |

10 年后护城河大概率仍在。安全边际主要来自「错杀」，而非低估值常态。

#### 芒格：失败模式
1. 永久高估值下的长期低回报  
2. 制造业 CAPEX 冰河期  
3. 竞品在中低端侵蚀（概率低于叙事）  

聪明人为什么不买？— 「贵」。

#### 李录
传感与测量是智能制造的「神经末梢」，文明级工具属性强。

#### 综合推荐
```
推荐度：★★★★☆
仓位类型：卫星（质量锚）
建议买入区间：更优=错杀至 PE≤32× 或 PS≤12×；现价可小仓位
建议持仓比例：占主题 25–30%
关键监测：订单/收入增速、毛利率是否下台阶、日元大幅波动
可替换：SMC（更便宜，ROE 更弱）
```

### 4.3 厦门钨业 — 材料卫星

#### 段永平
卖钨钼深加工产品与相关材料，本质是「有技术含量的资源加工」。好不好取决于加工溢价能否覆盖商品波动——2025 利润大增说明加工端有定价能力，但存货激增提醒周期风险。

#### 巴菲特：护城河

| 护城河 | 强度 | 证据 |
|--------|------|------|
| 品牌/定价权 | ★★★ | 深加工份额 |
| 转换成本 | ★★★ | 客户认证 |
| 网络效应 | ★ | 弱 |
| 规模效应 | ★★★★ | 一体化 |
| 资源/牌照 | ★★★★ | 钨产业链 |

#### 芒格：失败模式
1. 钨价暴跌 + 存货减值  
2. 出口管制伤及外销  
3. 能源材料业务拖累  

最坏情景：周期底部利润腰斩，TTM PE 被动抬升后杀估值（粗估回撤 30–50%，**估计**）。

#### 李录
关键金属深加工是大国竞争的实物底座；适合作为文明趋势下的周期卫星，而非永远持有的消费垄断。

#### 综合推荐
```
推荐度：★★★★☆
仓位类型：卫星（材料）
建议买入区间：现价可配置；存货周转恶化则等待
建议持仓比例：占主题 15–20%
关键监测：钨价、存货/收入比、硬质合金出口、稀土分部利润
```
