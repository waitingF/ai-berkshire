## 一、人物定位

**常士杉**（常总/私募一哥），实战派私募投资者，多次夺冠，杉涼资本创始人。

与巴菲特、芒格、段永平、李录的**本质区别**：

| 维度 | 四大师 | 常总 |
|------|--------|------|
| 核心身份 | 长期价值投资者 | 系统化交易者 + 宏观配置者 |
| 持有逻辑 | 买入好公司长期持有 | 好公司 + 好时机 + 好仓位 |
| 卖出纪律 | 逻辑变化才卖 | 板块破位无条件清仓，阶梯止盈 |
| 分析工具 | 商业模式、护城河、现金流折现 | 分配-周期-人性 + 量价 + 板块联动 |
| 宏观态度 | 段永平不看宏观 | 宏观×时代×流动性是核心变量 |

**结论**：常总不宜强行并入四大师框架作为"第五视角"，更适合作为**独立的战术/配置思维工具**（`/changzong-ask`）。

## 二、认知三层闭环

常总最核心的原创框架，贯穿全部资料：

```
分配之道（结构）→ 周期之道（节奏）→ 人性之道（动力）
```

### 2.1 分配之道

- 世界由**分配**驱动，非竞争驱动
- 三层模型：**规则（权力）→ 资源（资本）→ 定价（利润）**
- 利润天然向上游集中；平台是最强分配结构
- 高手看"谁在赚钱"，而非"谁在涨"

### 2.2 周期之道

- 周期 = 价值体系重新定价，非简单时间重复
- 三因素：**宏观（空间）× 时代（方向）× 流动性（动力）**
- 规则生成层：央行控利率、媒体塑叙事、平台控流动性
- 真正决定牛熊的往往是流动性，而非单一事件

### 2.3 人性之道

- 市场 = 人心总和
- 自利（损失厌恶、短期偏好）+ 社会性（从众、舆论）+ 认知偏差（确认偏误）
- 情绪周期：贪婪→恐惧→绝望→贪婪

## 三、交易执行体系

### 3.1 五步交易法（自上而下）

1. **宏观择时**：板块指数趋势确认，顺势而为
2. **微观选股**：龙头+潜龙，每板块2-3只
3. **风险隔离**：30/20原则（行业30%、个股20%）
4. **战术执行**：3221阵型 + 55%量能确认
5. **终局思维**：板块破位无条件清仓

### 3.2 3221阵型（仓位节奏）

| 步骤 | 比例 | 条件 |
|------|------|------|
| 底仓试探 | 35% | 结构临界点，验证逻辑 |
| 确认加仓 | 20% | 首笔浮盈 + 量能≥爆发日55% |
| 趋势强化 | 20% | 回调缩量、上涨放量 |
| 进攻仓 | 10-15% | 主升浪加速 |
| 现金储备 | 5-20% | 永远保留 |

**减仓体系**：盈利10-15%减30% → 15-30%再减20% → 30-50%阶梯撤退 → >50%清仓

### 3.3 三低买入法（逆向选股）

- 低价格（相对低估错杀）+ 低成交量（地量抛压衰竭）+ 低关注度（冷门洼地）
- 右侧确认：突破年线；刚性止损

### 3.4 超级牛股六大要素

赛道 + 叙事 + PEG增长 + 稀缺卡位 + 自由现金流 + 筹码共振

### 3.5 量价与技术面

- 55%量能法则、120日线生命线
- 量价四幕剧：建仓→洗盘→拉升→出货
- 期货导航：短线必看股指期货

### 3.6 恐慌套利模型

危机四阶段：观察(0-30%) → 初跌布局(30-50%) → 极端恐慌建仓(70-100%) → 修复止盈

### 3.7 五日线战法（短线起爆战术）

独立于3221的**5日均线锚定系统**，适合分歧洗盘后的突破捕捉：

| 环节 | 规则 |
|------|------|
| 入场三条件 | ①收盘价放量站上5日线 ②连续3日放量 ③7日内出现巨量（≥前日均量1.45倍） |
| 分歧确认 | 利好不涨、利空抗跌；分歧周期13~27个交易日 |
| 初次仓位 | ≤个股30% |
| 持有底线 | 收盘价不低于5日线下方7.5%（例：5日线10元→最低持有价9.25元） |
| 初级防守 | 跌破7.5%区间→减仓50%，设20%硬性止损 |
| 最终止损 | 回撤15%~20%→果断清仓 |
| 二次进场 | 止损后若放量站回5日线、不低于5日线下方4.5%、巨量企稳→最后一次分歧上车 |

**核心量化**：量价必须站上5日线；分歧越严重对个股越有利。

### 3.8 行业-低-势-量三维制胜（升级版三低）

将三低买入法扩展为完整操作框架：

```
投资成功 = 行业优质 + 买在低位(低) + 顺势持股(势) + 成交量验证(量)
```

| 维度 | 作用 | 关键信号 |
|------|------|---------|
| 行业 | 决定市场包容度，投资根本 | 颠覆性赛道 + PEG合理 |
| 低（价格） | 安全边际 | 历史估值低位、错杀非坏公司 |
| 低（量） | 抛压衰竭 | 地量区=主力建仓初期 |
| 低（关注） | 逆向筹码 | 冷门、被遗忘、低覆盖 |
| 势 | 赚大钱概率 | 高低点抬升、站稳60日线/周线中轨 |
| 量 | 验证势真假 | 势强量对=安全加仓；势强量错=假突破 |

**实战节点**：优质资产洗盘通常8-13周；第13周出现止跌K线+放量企稳→主升浪信号。

**量势铁律**：放量突破后5-10日维持爆发日55%以上→趋势成立；高位放量不涨+长上影→必须撤退。

### 3.9 起爆点与主力合力共振

大牛股诞生公式：**深度行业认知 × 产业资本 × 金融资本 × 技术形态突破**

三阶段演进：
1. **分歧奠基期**：低位门可罗雀，产业资本悄然吸筹（消灭信息差）
2. **共识加速期**：数据好转、研报覆盖，交易高手完成配置（消灭认知差）
3. **狂热高潮期**：大众信仰、论坛热议，赚钱效率最高但风险最大（消灭情绪差）

**临界引爆点**：放量突破120日线、洗盘极度缩量后再度放量创新高。

**通关四问**：①赛道是否未来三年天花板最高？②行内大佬是否在掏真金白银？③机构/ETF是否在进场？④形态弱转强是否确立？

**纪律**：不追高、不杠杆、单标的风险敞口受限、20%底线斩仓。

### 3.10 3221动态止盈（牛熊差异化）

在基础3221减仓体系上，按市场环境微调：

| 环境 | 策略 |
|------|------|
| 弱势/震荡市 | 盈利10-15%即兑现30-50%甚至全部仓位；利润回吐是最大敌人 |
| 强势牛市 | 盈利20%先减30%，之后每涨10-15%再减20%，保留10-30%底仓 |
| 底仓逻辑 | 成本压缩到极低后，不跌破盈亏平衡线就坚定持有——拿住5倍/10倍股的路径 |

### 3.11 100倍股七大共性（长期主义终极框架）

| 特征 | 内涵 |
|------|------|
| 时代主航道 | 不是行业龙头，是时代龙头（微软→PC，英伟达→AI，SpaceX→太空） |
| 超级痛点 | 雪中送炭而非锦上添花（解决巨大问题） |
| 巨大空间 | 收益率核心是市场天花板，非增长速度 |
| 极强护城河 | 最后卖的是生态（CUDA、AWS、社交网络） |
| 持续高研发 | 利润重新投入创新 |
| 死亡洗盘 | 无一100倍股一路上涨（亚马逊跌95%、英伟达回撤70%+） |
| 创始人梦想 | 吸引资本的是愿景，非利润 |

**时空五阶段**：种子期(0-3年) → 突破期(3-7年,~10倍) → 扩张期(7-15年,10-50倍) → 垄断期(15-25年,50-200倍) → 文明级(25年+,100-1000倍)

## 四、长期持有观（与四大师的最大交集）

常总也有"长期主义"一面，但是**有条件的长期**：

1. 锁定颠覆性赛道
2. 识别产业确定性拐点
3. 资金与心智双重容错（承受50-70%回撤）
4. 锚定业绩与现金流验证
5. 超越市场共识的高维认知

关键差异：即使长期持有，也执行**纪律减仓**（MU案例：420/460减仓），用55%量能判断是否继续持有。
