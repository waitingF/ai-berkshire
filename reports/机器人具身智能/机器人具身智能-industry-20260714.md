## 执行摘要

机器人投资必须拆成**两层生意**，否则会被叙事带偏：

| 层级 | 状态（截至2026-07） | 数据锚点 |
|------|---------------------|---------|
| **工业机器人 / 工厂自动化** | 成熟、可验证、盈利生意 | IFR：2024全球装机54.2万台；中国29.5万台占54% |
| **人形机器人 / 具身智能** | 商业化初期、出货已起量、盈利分化 | 高盛基准：2025约1.5–2万台 → 2026约5.1万台；宇树/智元/优必选已披露出货与收入 |

**核心判断（数据驱动）：**

1. **逻辑链前半段已部分验证**（劳动力短缺 + 端到端AI能力提升 → 工厂试点落地），但**后半段“百万台通用人形替代劳动力”仍属远期假设**，高盛基准2035年出货约140万台、TAM约$380亿——显著低于摩根士丹利等万亿叙事。
2. **当下最佳生意不在“整机概念股”，而在**：①已被验证的精密传动/运动控制；②已盈利的工业自动化龙头；③已被验证的垂直机器人（手术机器人 ISRG）。
3. **A股人形供应链存在严重“叙事—财报落差”**：多家媒体称三花/拓普锁定数十亿Optimus订单；但**三花智控2025年报仍写“研发、试制、迭代、送样”**，拓普年报亦称人形业务尚未显著贡献业绩。投资必须以年报为准。
4. **估值层面**：绿的谐波等纯正零部件标的已处历史极高PE分位；主题仓位应设硬上限，优先“能赚钱的机器人生意”，而非“讲故事的机器人概念”。

**行业总评：★★★☆☆（赛道方向对，但当前定价透支严重；适合分批、重质、控仓）**

| 维度 | 结论 | 信心度 |
|------|------|--------|
| 投资逻辑链（验证程度） | 工业自动化✅；人形工厂试点✅；规模化通用劳动替代⚠️ | **B** |
| 最佳环节（段永平“对的生意”） | 精密减速器/运动控制/手术机器人 | **A** |
| 最宽护城河（巴菲特） | Keyence传感、ISRG手术生态、Nabtesco RV | **A** |
| 最大风险（芒格） | 叙事泡沫 + 量产不及预期 + 零部件估值均值回归 | **A** |
| 文明趋势定位（李录） | 物理世界AI是文明级方向，但人形形态非唯一终局 | **B** |
| 整体估值水平 | 工业龙头合理偏贵；人形主题普遍贵 | **A** |

## 第一步：投资逻辑链构建与验证

### 1.1 逻辑链

```
底层趋势：AI从“屏幕智能”走向“物理智能”（端到端VLA/世界模型）
    → 劳动力短缺 + 危险/重复岗位ROI改善
        → 工厂/仓储愿意试点人形与移动操作机器人
            → 对关节执行器、减速器、丝杠、力觉、端侧算力产生增量需求
                → 供应链先受益（零部件放量先于整机盈利）
                    → 若泛化能力突破 → 跨场景通用人形放量
                        → 否则：退回“专用机器人 + 工业自动化”主线
```

### 1.2 逐环节验证

| 环节 | 核心假设 | 验证方式 | 验证结果 | 数据来源 |
|------|---------|---------|---------|---------|
| AI→物理智能 | 大模型可迁移到操控/导航 | 论文+产品演示+试点KPI | ⚠️ 试点有效，但泛化仍弱 | Figure BMW试点；宇树招股书风险披露 |
| 劳动力短缺→自动化需求 | 制造业招工难推升机器人ROI | IFR装机、各国空缺岗位 | ✅ 工业机器人十年翻倍 | IFR World Robotics 2025 |
| 工厂愿意用人形 | 人形可在不改造产线条件下上岗 | 真实产线工时/良率 | ✅ 有验证案例，但规模仍小 | Figure@BMW：1250+运行小时、9万+零件 |
| 试点→供应链订单 | 整机出货拉动减速器/执行器 | 公司年报分部收入 | ⚠️ **分化**：绿的谐波/鸣志有可见增量；三花/拓普年报仍“送样” | 各公司2025年报 |
| 出货→整机盈利 | 规模化后毛利转正 | 已上市整机财报 | ⚠️ 宇树已盈利；优必选仍亏7.9亿 | 宇树招股书；优必选港交所公告 |
| 通用人形替代劳动力 | 2030后大规模替代 | 第三方出货预测 | ❌/远期 高盛2035才到~140万台/年 | Goldman Sachs Research |

**但另一方面：** 媒体与卖方常把“框架意向/送样/产能规划”直接写成“锁定订单”。本报告对未在年报/招股书量化的订单一律标为**观点/传闻**，不作为投资依据。

## 第二步：产业链全景图

### 2.1 产业链结构

```
上游-核心零部件
  ├─ 精密减速器：谐波（绿的谐波/Harmonic Drive） · RV（Nabtesco/双环传动/中大力德）
  ├─ 传动丝杠：行星滚柱丝杠 · 滚珠丝杠（日本NSK/THK；国产五洲新春/北特等）
  ├─ 电机/驱动：无框力矩电机 · 伺服（汇川/鸣志/禾川；安川/松下）
  ├─ 传感：力/力矩、触觉、编码器、IMU（柯力等）；3D视觉（奥比中光）
  ├─ 端侧算力：NVIDIA Jetson/Isaac · 国产边缘SoC · MCU/FPGA
  └─ 结构件/散热/线束：汽车供应链迁移（三花热管理/执行器、拓普结构与执行器等）

中游-本体与子系统
  ├─ 工业机器人：FANUC / Yaskawa / ABB / KUKA(美的) / 埃斯顿 / 新松
  ├─ 协作机器人：Universal Robots(Teradyne) / 越疆(2432.HK) 等
  ├─ 人形/四足：Tesla Optimus · Figure · Agility Digit · Apptronik
  │              宇树 · 智元 · 优必选(9880.HK) · 傅利叶 · 银河通用 等
  ├─ 灵巧手/末端 · 移动底盘 · 关节模组（部分整机厂自研）
  └─ 软件栈：运动控制 · 仿真(Isaac/MuJoCo) · 具身大模型 · 远程运维

下游-场景
  ├─ 汽车/3C制造 · 仓储物流 · 危险作业
  ├─ 医疗手术（ISRG da Vinci）· 康复护理
  └─ 教育科研/文娱表演 · （远期）家庭服务

辅助
  ├─ 系统集成商 · 检测认证 · 保险与安全标准
  └─ 主题工具：中证机器人ETF（562500/159770等）· BOTZ/ROBO（海外）
```

## 第六步：文明趋势判断（李录框架）

1. **是否文明级范式？**  
   **观点：是“物理世界智能化”文明级趋势；但“双足人形”只是形态之一。** 轮式操作机器人、专用机械臂、AMR可能在多数场景ROI更高。

2. **历史类比：** 更接近“电力普及后的电机应用”或“PC后的嵌入式系统”，而非短期App泡沫。时间尺度是10–20年。

3. **10–20年终局：**  
   - 工厂：人机混线成为常态；  
   - 物流：移动操作自动化渗透；  
   - 家庭：仍可能是最慢一公里；  
   - 软件：具身模型+仿真数据闭环决定赢家通吃环节。

4. **赢家通吃更可能出现在：** 操作系统级具身模型、高端传感、手术等强监管垂直场景。  
5. **最可能被颠覆：** 只会做演示的整机品牌；缺乏软件的纯代工装配商。

**李录式追问：** 你下注的是“智能进入物理世界”，还是下注“一定是人形、一定是某家整机”？

理由：近端人形TAM相对全球股市体量仍小；多数上涨来自估值扩张而非自由现金流。
