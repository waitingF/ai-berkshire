## 一、超级趋势确认

```
趋势名称：创新药产业化扩张（非仿制药短缺叙事）
核心驱动力：ADC/多肽/CGT 从临床后期进入商业化，产能建设周期（3–5年）系统性落后于管线转化速度
已发生的验证事件（至少3个）：
  1. [2025全年] 中国创新药海外授权交易额突破约1300亿美元量级，ADC为绝对主角（行业媒体多源）
  2. [2025–2026] Lonza / 三星生物 ADC 商业化产线接近满负荷，新建产能最早约2027年释放（ByDrug/摩熵医药）
  3. [2025FY] 药明合联收入 RMB59.4亿（+46.7% YoY），全球ADC CRDMO份额>24%（公司年报/HKEX）
  4. [2025FY→2026指引] Bachem Building K 商业爬坡启动，2026年销售指引本地货币 +35%–45%（公司公告）
资本开支规模：全球生物药/CDMO/多肽扩产合计远超 $500亿/年量级（估计，含大药企自建+CDMO；精确TAM待行业白皮书交叉）
供需缺口判断：需求增速 > 供给扩产速度？是（高端偶联/高活载荷/商业肽合成/GMP质粒）
趋势确认：✅ 可追踪
```

**与仿制药短缺的区分（重要）**：USP/FiercePharma 报告的美国药品短缺主要是**仿制药 KSM/API 地理集中 + 低价采购**问题，属于另一条供应链。本扫描聚焦**创新药商业化制造咽喉**，不以仿制药短缺作为投资主线。

## 二、供应链物理拆解

```
Layer 0（终端）：已获批/即将获批的创新药（ADC、GLP-1肽、单抗、基因治疗）
Layer 1（充分定价）：原研药企（礼来/诺和诺德等）、大型综合 CDMO（Lonza/三星生物整体）
                 ⬆ 叙事充分，alpha 有限
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
                 ⬇ Layer 2–3 扫描区
Layer 2：
  ├─ ADC 商业化偶联（OEB5 高活偶联 + 制剂灌装）
  ├─ ADC Payload / Linker（高活毒素 API + 连接子）
  ├─ 商业级多肽 SPPS（GLP-1 及复杂肽 API）
  ├─ 病毒载体（AAV/慢病毒）商业产能
  └─ 高活无菌灌装 / 冷链
Layer 3：
  ├─ GMP 级质粒 DNA（pDNA）
  ├─ Protein A 配体 / 层析树脂
  ├─ 一次性生物反应器袋 / 无菌过滤器
  ├─ 特殊氨基酸 / 非天然砌块
  └─ 高活反应设备与隔离器认证产能
Layer 4：
  ├─ GMP 厂房许可 / 监管检查排期
  ├─ 高活化学人才
  └─ 临床患者招募（非物理硬瓶颈，本报告降权）
```

## 五、一页纸摘要（通过估值门槛的标的）

### 🎯 Bachem（BANB.SW）— 商业肽 SPPS 产能咽喉

**为什么是瓶颈**：GLP-1 与复杂肽商业化把固相合成产能打满；新建产能需 2–3 年，Building K 2026 才爬坡，供给被长约锁定。

**为什么是这家**：全球肽 CDMO 份额领先（行业估计约 20%+ 量级）；Building K 已过关键监管检查节点；2026 指引 +35–45% 直接把产能释放写进收入。

**催化剂**：
- 近期（1–3月）：2026-07-30 财报；Building K 商业产量爬坡数据
- 中期（3–12月）：2026 全年增速是否落在指引区间；Sisslerfeld/美国扩产进度

**主要风险**：
1. 大客户自建产能（礼来等）中期削弱外包
2. Capex 高企（2026 >CHF 4亿指引）压制 FCF（TTM FCF 已为负）
3. 多肽技术路线（液相/连续流）可能改变单位产能经济学

**关键数据**：市值 CHF 55.5亿 / 收入 CHF 6.95亿 / PS 8.0x / PE 37x / 2026指引 +35–45% / 肽业务纯正度高

**估值安全边际**：10年 25xPE 退出需净利约今天的 3.9 倍（CAGR ~14.5%）。结论：**弱安全边际，适合景气验证后深研，不宜叙事追高**。

**交叉验证**：✅ 客户/需求（代谢肽） / ✅ 收入验证 / ✅ 产能资本开支 / ⚠️ 估值偏不便宜

**结论**：**值得深入研究** → 建议 `/investment-team`（优先于同赛道亏损标的）

### 🎯 药明合联（2268.HK）— ADC 一站式偶联产能咽喉

**为什么是瓶颈**：缺的不是普通抗体产能，而是 OEB5 偶联 + 一体化交付；全球商业化 ADC 产能 2026–27 仍紧。

**为什么是这家**：全球份额 >24%（2025 收入口径）；收入 +46.7%；后期/商业化项目占比提升；新加坡基地 2026H1 GMP 放行意图明确；拟收购东曜补商业化产能（交易进度需跟踪）。

**催化剂**：
- 近期：2026-08-24 业绩；新加坡 GMP 放行进度；东曜要约交割状态
- 中期：2027 前后国际巨头新产能投放 → 观察利用率是否仍紧

**主要风险**：
1. **地缘/BIOSECURE 类政策**对中资 CDMO 订单迁移
2. 2027 产能潮可能阶段性过剩（摩熵等已提示出清风险）
3. 市值已约 USD 101 亿，略超小市值偏好；PS 12x 要求增速持续

**关键数据**：市值 HKD 792亿 / 收入 HKD 66.1亿 TTM / PS 12.0x / PE 50x（fwd ~34x）/ 增速 +47% / 瓶颈业务占比 ≈ 主业

**估值安全边际**：需净利 CAGR ~17.4%。结论：**景气验证充分，但价格已部分透支；回调或增速证伪时再加仓逻辑更强**。

**交叉验证**：✅ 客户（海外收入占比升至约 85%） / ✅ 收入 / ✅ 产能紧张叙事 / ⚠️ 估值黄灯 / ⚠️ 地缘

**结论**：**值得深入研究**（与 Bachem 并列第一档，但政策风险溢价更高）

## 六、反向验证（芒格式否定）

| 反向问题 | 证据取向 |
|---------|---------|
| 聪明人为什么不买？ | ADC/CXO 已炒多年；2027 产能投放 = 经典产能周期顶风险；中资 CDMO 政策折价 |
| 瓶颈能否被绕过？ | 定点偶联/新载荷可改配方，但 OEB5 物理约束难绕；肽可自建但 capex 与时间仍卡 |
| 中国能否快速复制？ | ADC 偶联国内玩家增多 → **S 级窗口可能短于 AI 材料瓶颈**；2027 后评级或降至 A |
| 终端需求腰斩？ | 单一适应症失败冲击大；GLP-1 竞争加剧可能压价格但量未必崩 |
| 管理层稀释？ | Bachem/合联未见极端增发叙事；皓元有转债需单独跟踪 |
| 估值隐含什么？ | 合联隐含持续 30%+ 增长数年；Bachem 隐含 Building K 成功爬坡 |

**观点（标注）**：创新药瓶颈的**时间窗口可能短于半导体材料**——因为药企与 CDMO 的 capex 响应更快，且监管一旦放行产能释放可同步。真正 alpha 更可能在 **2026–2027 产能仍紧、报表已兑现** 的阶段，而非 2028 之后。
