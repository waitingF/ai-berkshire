## 一、商业模式本质

### 1.1 核心生意定义

**事实**：合盛硅业从事工业硅、有机硅及多晶硅等硅基新材料的研发、生产与销售；自述为“我国硅基新材料行业中业务链最完整、生产规模最大的企业之一”（2025年报）。

**生意本质（观点，段永平语言）**：

> 这不是“卖品牌溢价”的消费生意，也不是“卖标准协议锁定客户”的平台生意，而是一门**把新疆低价煤电 + 硅石资源转化为工业硅分子，再继续加工成有机硅中间体/下游制品，并在景气时外售多余工业硅**的**重资产、高能耗、同质化大宗化工**生意。

赚钱公式可简化为：

```
利润 ≈ (产品市价 − 现金成本 − 折旧利息) × 销量 − 错误扩张的减值与停工损失
```

其中市价由行业供需决定；公司能主动控制的主要是现金成本与开工决策。光伏/多晶硅扩张把公式右端加上了一个“大额负项”（2025年减值约16.99亿元）。

## 二、平台/产品飞轮：纵向一体化如何运转

### 2.1 飞轮结构（事实描述）

```text
新疆低价煤 → 自备热电 → 低电价工业硅冶炼
        ↓                    ↓
   硅石/电极等自产配套    外售工业硅（合金/多晶硅客户）
        ↓
   内部供应有机硅单体合成（环体自用率极高）
        ↓
   硅橡胶/硅油等深加工 → 建筑/电子/汽车等下游
        ↓（失败分支）
   多晶硅/组件（2025停产/低利用 → 减值）
```

年报明确的协同逻辑（事实摘录要点）：
1. **原料稳定器**：工业硅保障有机硅/多晶硅原料；下游自用又稳定工业硅销售，支撑规模。
2. **沟通与规格协同**：内部供需优于外部采购，利于质量与成本。
3. **区位分工**：工业硅/多晶硅重仓新疆能源；有机硅布局浙江（贴近下游）+四川/新疆（热电联产）。

**关键运营数据支撑飞轮真实性**：
- 工业硅自用37.67万吨 / 产量145.47万吨 ≈ **26%** 内部消化；
- 环体硅氧烷自用约**94%**——有机硅飞轮高度“内循环”；
- 粗单体产量155.12万吨，与173万吨单体产能匹配度高，说明主链仍在转。

### 2.2 飞轮的正面与反面

| | 论据 |
|---|------|
| **正面：飞轮在底部仍转** | 2025工业硅均价跌约27%，公司工业硅毛利率仍有16.48%；年报称燃料动力占工业硅成本约35%量级且同比下降（媒体引年报：燃料动力占比35.39%、同比降26.86%）。一体化把“电价冲击”部分内部化。2026Q1毛利率回升至19.08%，经营现金流仍正。 |
| **反面：飞轮不是复利引擎，是波动放大器** | ①工业硅与有机硅高度同源，景气相关性高——“双主业”在周期底部同时承压（2025两者收入均双位数下滑），分散效果有限。②向光伏延伸本是飞轮“继续向下吃利润”，结果变成负飞轮：停产、减值16.99亿、上市首亏。③飞轮依赖新疆煤电政策与自备电厂稳定运行——年报风险章节自行提示煤价/电厂事故/监管可削弱成本优势。 |

### 2.3 “平台效应”检验（非共识）

市场常把一体化说成“平台”。按段永平/平台经济学严格标准：

| 平台特征 | 合盛是否具备 | 判定 |
|----------|--------------|------|
| 多边网络、交叉补贴吸引双边用户 | 否，上下游是内部转移定价 | 不是平台 |
| 边际获客成本趋零 | 否，增产=烧资本开支 | 否 |
| 用户产生内容/数据飞轮 | 否 | 否 |

**结论**：合盛拥有的是**纵向一体化成本飞轮**，不是互联网意义上的平台飞轮。把“煤电硅一体化”表述为平台，属于营销语言，不是商业模式事实。

### 2.4 本维度评分

| 维度 | 评分 | 结论 |
|------|------|------|
| 一体化飞轮有效性（降本/抗周期） | ★★★★ | 主业链条真实、可验证，底部仍有毛利 |
| 飞轮“复利/网络”属性 | ★ | 无网络效应；错误延伸会反向咬人 |

## 三、护城河分析（逐一验证）

段永平强调：护城河要能解释“为什么别人抢不走客户/利润”。对大宗材料，通常只剩**成本曲线位置**与**产能纪律**。

### 3.1 品牌 ★★

| | 内容 |
|---|------|
| **正面** | 年报称产品获“浙江名牌”等，是多晶硅/有机硅/硅铝合金龙头客户的重要供应商；有机硅下游分散，品牌有助于招标与长期供货名单。 |
| **反面** | 工业硅/DMC本质是规格品，采购决策权重是价格、纯度、交期、稳定性；品牌溢价空间窄。客户可用检测指标替代“品牌信仰”。 |
| **结论** | 存在“可靠供应商”声誉，但不是可提价的消费品牌护城河。 |

### 3.2 转换成本 ★★

| | 内容 |
|---|------|
| **正面** | 有机硅特定牌号（尤其电子/医疗/车规级）存在认证与配方适配，切换需重新验证；公司推进人体硅胶、医疗级硅橡胶等，有提高粘性的方向。前五大客户仅21%，分散反而说明锁定不强。 |
| **反面** | 大宗工业硅与通用DMC/硅橡胶转换成本低；价格战中客户可快速转向。2025价格普跌说明客户并未因转换成本而“锁价”。 |
| **结论** | 通用料转换成本弱；特种料在培育中，尚未构成公司级护城河。 |

### 3.3 网络效应 ★

| | 内容 |
|---|------|
| **正面** | 几乎无。 |
| **反面** | 第N个客户不会让第N+1个客户的产品更好用；相反，行业产能扩张是负和博弈。 |
| **结论** | 无网络效应。评分★1为框架下限。 |

### 3.4 规模效应 / 成本曲线 ★★★★（核心护城河）

| 证据类型 | 数据 | 含义 |
|----------|------|------|
| 产能规模 | 工业硅122万吨、有机硅单体173万吨，行业前列 | 固定成本摊薄、采购议价、排产灵活 |
| 产量份额（估算） | 2025产量145.47万吨 / 国内产量约420万吨 ≈ 35% | 龙头地位可验证 |
| 有机硅市场份额（第三方） | 观研报告：2025年1–11月有机硅中间体份额合盛约25.6%，CR6约69% | 集中度上升利于龙头 |
| 券商口径（需标注为二手） | 国泰海通等：工业硅产能行业占比约19%、有机硅单体约30% | 与年报“第一”叙事一致，精确占比以产能统计口径为准 |
| 成本动作 | 多数炉台单吨直接冶炼电耗降至万度以下；新疆自备电+煤 | 成本曲线左侧的可操作抓手 |
| 价格下跌中的毛利韧性 | 工业硅价-27%，毛利率仍16.48%（收窄9.62pct） | 相对同业高成本产能，仍有生存空间 |

**反面检验（A级研究必做）**：
1. **规模≠定价权**：份额再大，只要行业过剩，龙头也只能“少亏/微利”。2025归母亏损29.91亿已经说明。
2. **规模可能诱导过度投资**：同一套“做大做强”逻辑把公司推入光伏重资产，形成16.99亿减值——规模野心曾伤害股东回报。
3. **成本优势有地理与政策期权风险**：年报风险提示新疆煤价波动、自备电厂停产/处罚会削弱优势；双碳与能耗双控是长期悬剑。
4. **2026H1扣非仍为负**：预告归母盈利3.2–3.8亿，但扣非-0.33至-0.28亿——周期修复叙事下，核心经营利润的“干净程度”仍需观察（非经常项可能贡献扭亏）。

### 3.5 技术壁垒 ★★★

| | 内容 |
|---|------|
| **正面** | 完整掌握工业硅—有机硅—多晶硅工艺；研发费用2025约5.21亿元（东方财富转载）；氨基硅油/乳液产业化、流化床改造、碳化硅6/8英寸衬底量产、12英寸送样——技术纵深在加深。 |
| **反面** | 主业赚钱环节仍是成熟工艺的规模制造，非专利垄断；碳化硅尚处投入期，对2025/2026利润贡献未在分部收入中形成独立支柱；多晶硅技术未能阻止停产与减值。 |
| **结论** | 有工程与工艺壁垒，足以拉开与小厂差距；不足以形成类似软件/创新药的“技术定价权”。 |

### 3.6 护城河总表

| 护城河类型 | 评分 | 是否成立 |
|------------|------|----------|
| 品牌 | ★★ | 弱成立（供应商声誉） |
| 转换成本 | ★★ | 弱成立（仅特种料） |
| 网络效应 | ★ | 不成立 |
| 规模/成本曲线 | ★★★★ | **主要护城河** |
| 技术壁垒 | ★★★ | 部分成立（工艺+高端延伸） |
| **综合护城河** | ★★★ | 成本领先型护城河，厚度中等、耐久依赖能源区位与资本纪律 |

## 四、用户/客户价值

### 4.1 客户是谁、买什么

| 下游 | 买什么 | 独特价值（相对小厂） | 可替代性 |
|------|--------|----------------------|----------|
| 有机硅深加工厂/制品厂 | DMC/环体/硅橡胶/硅油 | 大规模稳定供货、牌号齐全、内循环保障交期 | 高（通用料） |
| 铝合金/金属硅用户 | 工业硅 | 成本与品质稳定、大单交付能力 | 高 |
| 光伏产业链 | 工业硅/多晶硅（历史） | 一体化原料；但2025自身多晶硅停产，对客户价值收缩 | 极高，且行业过剩 |
| 高端/新兴（医疗、电子、碳化硅） | 特种硅橡胶、SiC衬底/粉料 | 性能与认证（培育中） | 中低（若认证成功） |

### 4.2 独特价值的诚实表述

**事实支撑的独特价值**：
1. **可靠的大规模供给**——在工业品里，“不断供、能降本”本身就是价值；
2. **一体化带来的成本与交期**——环体94%自用说明公司首先是自己的最佳客户，再对外销售边际产量；
3. **客户结构分散**——前五大仅21%，对单一客户依赖度低（对合盛是风险分散；对客户议价则意味着合盛也难“卡脖子”）。

**反面**：
- 客户获得的是**更便宜、更稳的大宗投入品**，不是不可替代的系统；
- 当行业价格崩塌时，客户价值更多体现为“合盛还能开工”，而非“客户愿意多付钱”；
- 光伏客户侧：公司自身减值与停产说明向下游延伸并未增强客户粘性，反而消耗资本。

### 4.3 本维度评分

| 维度 | 评分 | 结论 |
|------|------|------|
| 客户独特价值 | ★★★ | 价值真实但同质；核心是成本与供给可靠性 |
| 客户锁定/定价能力 | ★★ | 分散客户+同质产品 → 定价权弱 |

## 五、业务矩阵与协同效应

### 5.1 业务矩阵

| 业务 | 战略角色 | 2025状态 | 协同方向 |
|------|----------|----------|----------|
| 工业硅 | 成本锚 + 外售现金 | 价跌量承压，毛利仍正 | →有机硅原料；→合金/光伏原料 |
| 有机硅 | 利润与深加工延伸 | 收入第一，毛利受压后Q4修复 | ←吃内部工业硅；→特种料 |
| 多晶硅/光伏 | 曾经的成长故事 | 停产、减值、计划收缩 | 与工业硅协同逻辑成立，资本回报失败 |
| 碳化硅等新材料 | 期权 | 技术进展披露多，收入贡献未成支柱 | 材料体系延伸，尚非现金引擎 |
| 煤电/矿产配套 | 成本基础设施 | 自备电+硅石资源 | 支撑工业硅成本曲线 |

### 5.2 协同：什么成立、什么不成立

**成立的协同（事实）**：
- 工业硅→有机硅的原料与质量闭环（高自用率）；
- 新疆能源→工业硅成本；
- 有机硅深加工消化环体，平滑单体波动（有限）。

**不成立或弱协同（反面）**：
- **工业硅与有机硅的需求对冲弱**：二者常同步走弱（2025双降）；
- **光伏协同变成价值毁灭**：技术上“有协同”、财务上“负协同”；
- **碳化硅协同偏远期**：与当前工业硅/有机硅现金引擎关联弱，不可计入近端护城河。

### 5.3 本维度评分

| 维度 | 评分 | 结论 |
|------|------|------|
| 主业协同（硅—有机硅—能源） | ★★★★ | 清晰且被运营数据验证 |
| 扩张协同（光伏） | ★ | 战略协同叙事失败，资本配置扣分 |
| 业务矩阵健康度 | ★★★ | 收缩光伏、聚焦主业是正确纠偏方向（观点） |

## 六、段永平“好生意”标准评估

段永平常用尺子（本报告采用的可操作定义）：
1. **差异化**：产品是否“不一样”，客户是否在乎；
2. **定价权**：涨价是否丢客户；
3. **可持续竞争优势**：十年后是否仍难被追上；
4. **本分/聚焦**：是否做能力圈内且懂的事；
5. （隐含）**长期回报质量**：不只看份额，看是否持续创造自由现金与高ROE。

### 6.1 逐项评估

| 标准 | 评分 | 证据与判断 |
|------|------|------------|
| 差异化 | ★★ | 主产品为规格品；特种有机硅/SiC在提升差异化，尚未主导收入 |
| 定价权 | ★ | 2024–2025价格由供需决定，公司为价格接受者；“反内卷/协同减产”是行业行为，不是单企业提价权 |
| 可持续竞争优势 | ★★★ | 成本曲线左侧+产能规模有持续性，但依附资源区位与政策，且周期会阶段性抹平利润 |
| 本分/聚焦 | ★★ | 硅基主业本分；跨入光伏并形成巨额减值，偏离“只做懂的、能赢的” |
| 客户愿付溢价 | ★★ | 溢价空间窄，更多是成本优势转化为份额与开工率 |
| **好生意综合** | ★★ | **不是段永平经典意义上的好生意**；更接近“周期底部可研究的成本龙头特殊机会” |

### 6.2 与市场共识的刻意分歧（非共识）

| 市场常见表述 | 本报告检验 |
|--------------|------------|
| “双龙头护城河深厚” | 护城河主要是成本，不是品牌/网络；深度中等 |
| “一体化平台公司” | 是制造一体化，不是平台 |
| “2026扭亏=基本面反转完成” | H1预告扣非仍亏，需区分周期价格修复 vs 主业真实赚取现金利润 |
| “光伏只是短期阵痛” | 对股东而言已是真实财富损失（减值），显示资本配置短板 |
| “规模越大越好” | 规模在过剩行业可加速出清对手，也可诱导自身过度扩张 |

### 6.3 段永平式一句话（观点）

> 合盛懂怎么把电和矿做成硅，也证明了在价格暴跌时还能靠成本活着；但它卖的不是“爱不释手”的产品，而是跟着周期呼吸的分子。你是在买**成本曲线上的位置**和**管理层下次还犯不犯光伏那种错**，不是在买一门可以无视价格的好生意。

## 八、该维度总体结论

### 8.1 总体结论（给团队汇总用）

1. **生意是什么**：合盛是全球领先的工业硅+有机硅一体化制造商，利润来自“市价−成本”的价差，而非品牌溢价或网络租金。  
2. **护城河是什么**：经逐项证伪后，唯一坚实的护城河是**新疆煤电硅一体化支撑的成本曲线左侧位置 + 产能规模**；品牌、转换成本、网络效应均弱。  
3. **用户价值是什么**：为下游提供大规模、较稳定、成本有竞争力的硅基投入品；独特但不“不可替代”。  
4. **飞轮怎么转**：工业硅自用+环体高度内循环构成真实制造飞轮；向光伏的延伸在2025年证明飞轮可以反向摧毁利润。  
5. **段永平标准下的定位**：**不是经典好生意**（差异化与定价权不足），而是**可理解的周期成本龙头**。投资含义上，更依赖周期位置、供给出清与资本开支纪律，而非“买入并永远持有式”的商业模式红利。  
6. **反面仍成立的核心风险（商业模式层）**：能源区位优势变动、行业长期过剩、主业高相关导致对冲失效、管理层再次跨界扩张。

### 8.2 置信度分层

| 内容 | 置信度 |
|------|--------|
| 收入结构、产能、价格、客户集中度、一体化描述 | 高（年报原文） |
| 成本曲线相对位置、份额百分比精确值 | 中（部分依赖行业协会/第三方/券商口径） |
| “2026年后护城河将增强/减弱”的路径判断 | 低（取决于价格、政策、资本纪律） |
| 碳化硅成为第三成长曲线 | 低（进展有，经济贡献未验证） |

### 8.3 对 Team Lead 的交接要点

- 不要把“龙头+一体化”直接写成买入理由；财务与估值同事需单独检验：**周期中枢利润**与**扣非盈利质量**（注意2026H1扣非仍负）。  
- 行业同事应重点验证：工业硅/有机硅供给出清是否可持续，而非仅引用公司减产会议新闻。  
- 风险同事应覆盖：新疆能源政策、自备电厂、家族治理与再扩张冲动（光伏覆辙）。
