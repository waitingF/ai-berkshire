# 合盛硅业（603260.SH）行业竞争分析 — 芒格视角

**角色**：行业研究员  
**分析框架**：查理·芒格（逆向思考 / 竞争格局 / 激励与系统）  
**数据截止**：2026-07-19  
**信息丰富度评级**：A级（重点反面检验与非共识）  
**研究方法**：WebSearch 联网检索 + 公司年报/行业协会/第三方行情交叉验证  
**核心警示**：警惕把「产能龙头」自动等同于「好投资」

---

## 芒格式一句话

> 反过来想：在工业硅名义产能近千万吨、有效产能利用率不足五成、多晶硅需求同比大幅收缩的环境下，**「全球第一产能」首先是沉没成本与周期杠杆，其次才是护城河**；有机硅的「反内卷」抬价可以改善利润，但不能消除大宗商品定价权缺失这一商业本质。

## 1. 行业规模与增长

### 1.1 工业硅：规模巨大，2025年首次近五年需求负增长

| 指标 | 2025年数据 | 同比/说明 | 来源 |
|------|-----------|----------|------|
| 全国总产能 | 980.57万吨 | 名义产能高位 | Mysteel《2026年工业硅市场挑战机遇并存》2026-01 |
| 有效产能 | 约860万吨 | 规划投产兑现率不足七成 | 同上 |
| 产量 | 402.5–405.5万吨 | 约-14%～-14% | Mysteel / CPIA |
| 表观消费 | 334.7万吨 | **-16.6%，近5年首次负增长** | CPIA/中国光伏行业协会 |
| 出口 | 72.0–72.1万吨 | 基本持平（约-0.6%） | 合盛年报转述海关；CPIA |
| 产能利用率（粗算） | 产量/有效产能 ≈ **47%** | 观点：结构性严重过剩 | 由上表推算 |

**下游需求结构（2025，CPIA）**：

| 下游 | 耗硅量 | 占比 | 同比结构变化 |
|------|--------|------|-------------|
| 多晶硅 | 171.0万吨 | **45%** | 占比-10ppt（仍最大） |
| 有机硅 | 128.8万吨 | **34%** | 占比+7ppt |
| 铝合金 | 77.5万吨 | **20%** | 占比+3ppt |

**渗透率/成熟度判断（事实+观点分离）**：

- **事实**：工业硅是成熟基础材料，不存在「从0到1」的渗透故事；增长绑定光伏装机、有机硅消费与铝硅合金。
- **观点**：2025年需求负增长说明上一轮光伏驱动的「成长叙事」已阶段性失效；2026年若多晶硅继续自律减产，工业硅更像**周期商品**而非成长赛道。

### 1.2 有机硅：全球产能向中国集中，需求仍有温和增长

| 指标 | 数据 | 来源 |
|------|------|------|

| 中国占全球产能 | 约70%–75%（中间体/聚硅氧烷口径略有差异） | 国海；CAFSI/公司公告转述 |
| 2024年中间体产量 | 225.3万吨 | 智研咨询 |
| 2025年产量/表观消费（华经口径） | 产量242.79万吨；表观消费196.15万吨 | 华经《2026年中国有机硅行业供需现状》 |
| 2025聚硅氧烷产量（ACMI/SAGSI） | **276.6万吨，+9.2%** | 合盛2025年报 |

| 需求增速展望 | 国海：2025–2027表观消费增速约8.0% / 7.0% / 8.8%；ACMI/SAGSI：2026消费增速约**5%–8%** | 国海；合盛年报 |

**渗透率结构（事实）**：建筑密封胶等传统领域占比下滑；光伏用胶、新能源车密封/导热、电子封装、AI液冷等新兴领域贡献增量（东岳年报及证券日报2026-04报道）。这是**结构升级**，不是整体爆发。

### 1.3 价格：两条曲线、两种周期位置

| 品种 | 2025关键点位 | 2026上半年动态 | 来源 |
|------|-------------|----------------|------|
| 工业硅 | 6月期货一度跌破7000元/吨；金属硅553均价同比约**-27%** | 期货机构预期下半年8300–9000元/吨宽幅震荡 | Mysteel；百川盈孚（合盛问询/媒体转述）；国联期货2026-06 |
| 有机硅DMC | 年均约1.2万元/吨附近；低点约10200元/吨 | 反内卷后：2026-01约14000；2026-04末约**14700**元/吨（较2025-11约+27%–35%） | Mysteel；国海；生意社；澎湃/东岳报道 |

**芒格提醒**：价格修复≠护城河变宽。工业硅仍在成本线附近博弈；有机硅涨价高度依赖行业协同减产，属于**激励相容的脆弱均衡**。

## 2. 竞争格局总览

### 2.1 工业硅：产能分散，「一超」但未「寡占」

| 集中度指标 | 数值 | 含义 | 来源 |
|-----------|------|------|------|
| CR5（产量/市占口径） | **约34%** | 前五合计不足四成 | Mysteel 2026-01 |
| CR10 | **约40%–44.6%** | 仍属中低集中度 | Mysteel；CPIA（CR10=44.6%，较2024+5.3ppt） |
| CR30 | 约55% | 长尾大量中小炉 | Mysteel |
| 区域 | 新疆产量占全国 **52.8%**；内蒙跃升第二大产区 | 供给向西北煤电/资源区集中 | Mysteel |

**合盛位置（事实）**：

| 项目 | 数据 | 来源 |
|------|------|------|
| 工业硅产能 | **122万吨/年**（截至2025末） | 合盛2025年报 |
| 产能占比（相对国内总产能） | 约**12%–19%**（取决于用名义/有效产能分母；券商常用约19%） | 国泰海通跟踪报告摘要；年报产能 |

> **反面检验**：名义产能第一 ≠ 定价权。CR5仅34%说明合盛无法单方面锁价；价格由边际高成本产区开工、多晶硅采购与期货定价共同决定。

### 2.3 竞争策略对比矩阵

| 维度 | 合盛 | 东岳/新安/兴发等第二梯队 | 中小产能 |
|------|------|--------------------------|---------|
| 成本曲线位置 | 最左端（新疆自备电+自给工业硅） | 中左至中（部分外购硅/电网电） | 右侧（枯水期西南、小炉型） |
| 产品定位 | 大宗中间体为主，加速下游与高端牌号 | 深加工、差异化牌号更强（如新安全系列） | 同质化中间体/区域市场 |
| 周期策略 | 高开工抢份额、现金流回笼 | 减产挺价参与「反内卷」 | 停产/退出 |
| 资本配置风险 | 光伏跨界留下巨额闲置资产 | 相对克制或另有主业对冲 | 生存优先 |
| 定价权 | 行业最大影响者之一，但仍非垄断 | 协同定价参与者 | 价格接受者 |

## 3. 核心竞争者威胁评估（逐个）

评分说明：威胁度★1–5（对合盛长期经济利润的侵蚀能力）；越高越危险。

### 3.1 东岳硅材（300821.SZ）— 威胁度 ★★★★☆

| 项目 | 评估 |
|------|------|
| 规模 | 单体约60万吨，中间体份额约10%，第二梯队领头 |
| 优势 | 硅粉到深加工一体化；2025开发新牌号80+；「反内卷」后盈利弹性已兑现（媒体报道2026单季盈利显著修复） |
| 对合盛含义 | 在有机硅定价与减产联盟中是**关键协同方**；若其深加工溢价持续，合盛「中间体大宗化」路径相对吃亏 |
| 但另一方面 | 成本曲线未必优于合盛新疆体系；规模仍约为合盛单体产能的1/3 |

### 3.2 兴发集团（600141.SH）— 威胁度 ★★★☆☆

| 项目 | 评估 |
|------|------|
| 规模 | 单体约60万吨 + 规划增量（投产时点不确定） |
| 优势 | 磷矿/磷化工协同，原料与园区成本；券商重点标的之一 |
| 对合盛含义 | 潜在供给增量的主要不确定性来源；若规划落地，会拉长有机硅过剩消化时间 |
| 但另一方面 | 行业低利润抑制投产意愿（国海：2026–2027主要可见增量或为新疆其亚等少数项目） |

### 3.3 新安股份（600596.SH）— 威胁度 ★★★☆☆

| 项目 | 评估 |
|------|------|
| 规模 | 单体55万吨；工业硅现有+规划约30万吨 |
| 优势 | 少有的硅橡胶/硅油/硅树脂/硅烷偶联剂全覆盖；下游附加值结构优于纯中间体商 |
| 对合盛含义 | **产品结构威胁**大于规模威胁——合盛若不能把利润池向上游大宗之外迁移，长期ROE会被「深加工型」对手压低 |
| 但另一方面 | 草甘膦周期与硅基周期叠加，财务波动大；规模仍显著小于合盛 |

### 3.4 鲁西化工（000830.SZ）— 威胁度 ★★★☆☆

| 项目 | 评估 |
|------|------|
| 规模 | 单体约45万吨 |
| 优势 | 化工园区一体化；在2025年底行业协调中扮演推动角色（媒体报道） |
| 对合盛含义 | 协同定价的组织者之一；本身不是最大产能威胁 |
| 但另一方面 | 若协同破裂，鲁西也可能率先降价抢量（激励问题） |

### 3.5 蓝星星火 / 陶氏等外资系 — 威胁度 ★★★☆☆

| 项目 | 评估 |
|------|------|
| 优势 | 高端牌号、工艺与客户认证；陶氏2025-07宣布关闭欧洲约14.5万吨DMC产能（中研普华等转述）显示**海外退出** |
| 对合盛含义 | 短期利好中国出口替代；长期在高端电子/医疗硅材料上仍压制合盛溢价能力 |
| 但另一方面 | 海外能源成本高，大规模回流概率低 |

### 3.6 东方希望等新疆/一体化工业硅巨头 — 威胁度 ★★★★☆

| 项目 | 评估 |
|------|------|
| 优势 | 同样占据西北成本曲线左端；工业硅赛道集中度低，价格战武器充足 |
| 对合盛含义 | **工业硅侧真正的同质竞争者**；合盛的市占优势可被同等成本的大厂对冲 |
| 但另一方面 | 多数对手缺乏合盛级别的有机硅消纳能力，周期底部抗压略弱 |

### 3.7 通威/协鑫/大全等光伏硅料巨头（外溢威胁）— 威胁度 ★★★★☆（对工业硅需求）

| 项目 | 评估 |
|------|------|
| 机制 | 多晶硅产量2025约133万吨（合盛年报转硅业分会，同比-27.8%），直接砍掉工业硅最大需求块 |
| 对合盛含义 | 不是抢合盛客户，而是**摧毁需求曲线**；合盛自有5万吨多晶硅停产，既是受害者亦是错误跨界的注脚 |
| 但另一方面 | 光伏「反内卷」若抬升硅料开工，工业硅需求可能阶段性回补——但2026装机展望偏谨慎（合盛年报引用行业协会展望） |

### 3.8 西南水电硅（云南/四川中小炉）— 威胁度 ★★☆☆☆（枯水期）/ ★★★☆☆（丰水期）

| 项目 | 评估 |
|------|------|
| 事实 | 2025云南产量同比-53%，四川-12.5%（Mysteel）——枯水与亏损已出清部分产能 |
| 风险 | 丰水期电价可低至约0.20–0.28元/kWh，完全成本或至8000–10000元/吨，短期冲击价格（新浪财经大宗观察2026-06） |
| 芒格解读 | 这是**季节性边际供给**，不是永久性竞争壁垒；但会阻止工业硅价格出现「钢铁供给侧」式的单边大牛市 |

## 4. 各细分赛道格局

### 4.1 工业硅赛道

```
供给：名义产能980万吨 >> 有效860万吨 >> 产量~405万吨
需求：多晶硅45% + 有机硅34% + 铝合金20%（2025 CPIA）
价格形成：成本底（西北自备电） vs 丰水期边际供给 vs 期货引导
出清状态：进行中但缓慢——行政出清弱于市场化亏损出清
```

| 评分项 | ★ | 说明 |
|--------|---|------|
| 行业吸引力（芒格「是否值得待」） | ★★☆☆☆ | 过剩+弱定价权 |
| 合盛相对竞争力 | ★★★★☆ | 成本与规模领先 |
| 竞争强度 | ★★★★★ | 分散、同质、高固定成本 |

### 4.2 有机硅单体 / 中间体

| 评分项 | ★ | 说明 |
|--------|---|------|
| 行业吸引力 | ★★★☆☆ | 扩产尾声+需求5%–8%增长，优于工业硅，但仍周期 |
| 合盛相对竞争力 | ★★★★☆ | 份额第一+原料自给 |
| 竞争强度 | ★★★★☆ | CR8高，但「反内卷」前曾长期价格战 |

**关键非共识**：产能集中本应带来定价权，但2021–2024集体扩产证明——**集中度高的行业仍可集体自杀**。2025Q4起的协同是对前期错误的纠正，不是新制度经济学意义上的稳定卡特尔。

### 4.3 有机硅下游制品（硅橡胶/硅油/硅树脂）

| 特征 | 数据/事实 | 含义 |
|------|----------|------|
| 硅橡胶产能利用率 | 2023年整体约57%（长江证券转ACMI） | 制品端也过剩 |
| 竞争者结构 | 合盛有制品但偏大宗；新安/东岳/硅宝/回天等在细分更深 | 利润池更分散在下游品牌与配方 |
| 合盛进展 | 氨基硅油、乳液产业化；电子级凝胶等（年报） | 方向正确，体量贡献尚不确定（低置信度） |

### 4.4 多晶硅 / 光伏相关（对合盛：战略收缩赛道）

| 项目 | 事实 |

## 6. 产业链分析：上中下游价值分配

```
上游                    中游（合盛主战场）              下游
硅石/煤/石油焦     →    工业硅（冶炼）          →    多晶硅/组件（光伏）
电力（占成本大头）  →    有机硅单体/DMC         →    硅橡胶/密封胶/电子胶
氯甲烷/甲醇        →    环体/硅油              →    建筑/汽车/日化/医疗
```

### 6.1 价值分配表（定性为主，标注置信度）

| 环节 | 经济利润特征（2025–2026H1） | 议价力 | 置信度 |
|------|------------------------------|--------|--------|
| 电力/能源 | 容量电价与煤价使成本上移；自备电仍优于外购 | 政策与煤电合同决定 | 中 |
| 工业硅冶炼 | **行业整体微亏至盈亏平衡**；仅左端成本产能勉强正毛利 | 弱（CR低） | 高 |
| 有机硅中间体 | 2025深度承压；2026协同后利润修复至「可活」 | 中（CR高但协同脆弱） | 中高 |
| 有机硅深加工/品牌制品 | 分化大：配方与认证决定溢价 | 中高（细分龙头） | 中 |
| 多晶硅 | 2025巨亏出清；政策脉冲后仍高库存 | 弱–中 | 高 |
| 终端（车/电子OEM） | 压价采购硅胶辅材 | 强 | 中 |

### 6.2 合盛在链上的位置

**事实**：合盛吃的是「上游能源禀赋 + 中游规模制造」的差价，并通过内部工业硅→有机硅消纳（2025工业硅自用37.67万吨）降低外售波动。

**观点（芒格）**：

1. 价值最丰厚的长期位置往往在**不可替代的能源契约**与**下游配方/认证**两端；合盛强在前者、弱在后者。
2. 向多晶硅延伸本想捕获光伏红利，结果证明中游制造在过剩行业没有「一体化溢价」，只有「一体化亏损」。
3. 「煤电硅」护城河的最大敌人不是同行广告，而是**电价制度变迁**与**绿电/碳约束重估成本曲线**。

## 8. Bull vs Bear（行业与竞争维度）

### 看多逻辑（需证据）

1. 有机硅扩产周期结束，CR8>80%，协同减产后DMC自低点反弹超25%（百川/生意社）。
2. 工业硅向西北集中，小炉与枯水期西南产能出清，CR10提升（CPIA）。
3. 工信部节能监察抬高合规门槛，利好大炉龙头。
4. 海外有机硅退出（陶氏欧洲关停等），中国出口替代空间打开。
5. 合盛工业硅自给+自备电，在价格战中存活率最高。

### 看空 / 反面检验（A级重点）

1. **产能龙头陷阱**：工业硅有效产能利用率约47%，龙头高开工可能是「以亏损换份额+现金流」而非护城河变宽（行情通等市场分析有此逻辑，需与年报现金流交叉看）。
2. **光伏外溢未结束**：多晶硅仍占耗硅45%，装机回落则工业硅难以单靠有机硅/铝合金撑起景气。
3. **电价中枢上移**：容量电价使新疆成本优势「相对」收窄，不是消失，但弹性变小。
4. **出口退税取消**：初级聚硅氧烷竞争力受损，大宗出口路径变窄。
5. **协同定价脆弱**：卡特尔式减产依赖激励；一旦库存回升或丰水期冲击，价格可快速回吐。
6. **深加工落后于第二梯队**：份额第一主要在中间体，附加值池可能被新安/东岳等分流。

## 10. 总体结论

合盛硅业在工业硅与有机硅两条赛道上均占据**全球/全国产能第一**与**成本曲线左端**，竞争地位客观强于绝大多数同行；有机硅「一超多强」的格局与扩产尾声，使其中期竞争环境优于高度分散的工业硅。

但芒格视角的关键结论是：**规模与成本优势解决的是「谁最后破产」问题，不能自动解决「股东能否获得满意回报」问题。** 2025年工业硅需求近五年首次负增长、多晶硅耗硅占比仍高达45%、有效产能利用率不足五成，说明行业仍处供给侧出清中段；有机硅价格修复高度依赖「反内卷」协同与出口/深加工结构升级，且面临取消出口退税、甲醇成本扰动与电价制度抬升等逆风。

对投资团队的含义（供 team-lead 汇总，非投资建议）：

- 竞争分析维度综合评分：**★★★☆☆（3/5）**
- 可确认的正面：成本护城河真实、有机硅格局优于工业硅、政策能耗约束偏利龙头
- 必须保持的负面：产能龙头≠定价权；光伏外溢与高固定成本使周期杀伤力极大；下游价值分配并不偏向合盛当前的大宗定位

> 「反过来想，总是反过来想。」——在喊「硅业龙头发反弹」之前，先回答：若2026下半年工业硅在8300–9000元/吨震荡、有机硅协同松动，合盛的成本优势是否仍足以覆盖债务与闲置光伏资产的拖累？这不是行业研究能单独闭环的问题，但必须作为竞争结论的边界条件写下。
