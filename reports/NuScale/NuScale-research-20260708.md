## 二、生意本质分析 — 段永平"对的生意"

### 这门生意的本质

NuScale 卖的不是电，而是**核反应堆设计 IP + 监管许可 + 工程服务**——类似"卖铲子"给核电开发商，但铲子的 FOAK（首台套）成本已被 CFPP 项目证伪。

### 商业模式画布

| 维度 | 内容 |
|------|------|
| 客户 | 公用事业（TVA）、欧洲政府（RoPower/罗马尼亚）、数据中心（ENTRA1/Standard Power） |
| 价值主张 | 唯一 NRC 认证 SMR 设计；77 MWe 模块可组合至 924 MWe |
| 收入模式 | NPM 销售 + 技术许可 + 工程服务 + 未来燃料/O&M（均未规模化） |
| 关键资源 | NRC 设计认证（50 MWe + 77 MWe SDA）、513 项专利 |
| 关键伙伴 | **ENTRA1（全球独家商业化）**、Fluor（EPC）、Doosan（制造） |
| 成本结构 | R&D ~$45M/年 + ENTRA1 里程碑现金流出 |

### 5年盈利能力趋势

```
净亏损($M):  -102 → -26 → -58 → -137 → -356
收入($M):      3 →   12 →  23 →   37 →   31
```

**趋势判断**：收入 2024 见顶后回落；2025 亏损暴增主因 ENTRA1 Milestone 1（$507M 非现金费用 + Q1'26 $260M 现金支付），非经营恶化 alone。但 Q1'26 收入崩塌至 $0.6M 表明**工程服务管道已枯竭**。

### 毛利率与同行

| 公司 | 毛利率 | 说明 |
|------|--------|------|
| NuScale FY2024 | 86.7% | 工程服务高峰，不可持续 |
| NuScale FY2025 | 36.3% | 回归常态 |
| NuScale Q1'26 | 3.7% | 几乎无收入 |
| GE Vernova Nuclear | ~25–30%（估计） | 成熟核电设备商 |

高毛利来自**轻资产工程咨询**，而非模块制造——模块销售毛利率尚未验证。

### 经营杠杆

NuScale 固定成本（R&D + G&A ~$100M+/年）相对当前 $19M TTM 收入极高。**负经营杠杆**：收入每降 1%，亏损放大。ENTRA1 里程碑支付进一步放大现金流出。

### 段永平式追问

> **这门生意好在哪？如果只能用一句话描述，是什么？**

**正面**：全球唯一获 NRC 标准设计批准的商业 SMR，18 年+$18 亿研发投入形成的监管壁垒真实存在；AI 数据中心 24/7 基载电力需求是文明级趋势。

**反面**：CFPP 项目证明 FOAK 经济学不成立（LCOE 从 $58/MWh 飙至 $89/MWh 后取消）；公司**不控制商业化渠道**（ENTRA1 独家）；18 年零模块销售——"对的生意"尚未被商业验证。

**一句话**：一门**监管期权生意**——赌 SMR 时代到来且 NuScale 设计被选中，但目前仍是"有证无单"。

### 护城河趋势判断

- **过去5年**：NRC 50 MWe 认证（2023）→ 77 MWe SDA（2025），监管护城河加宽；但 CFPP 取消（2023）严重损害商业信誉
- **未来5年预判**：若 RoPower 2033 首台并网成功，护城河显著加宽；若 TVA/ENTRA1 框架不转绑定，护城河仅为"纸面认证"

### 巴菲特式追问

> **10年后这条护城河还在吗？什么能摧毁它？**

**还在的条件**：至少 2–3 个 VOYGR 电厂并网运营，积累运行数据与供应链规模效应。

**摧毁路径**：
1. RoPower 或 TVA 项目再次因成本超支取消（CFPP 2.0）
2. 竞争对手 FOAK 电厂先并网且 LCOE 更低
3. ENTRA1 无法融资，NuScale 被独家协议锁死却无法商业化
4. 新一代核技术（熔盐堆、快堆）跳过 LWR SMR 阶段

## 六、行业与文明趋势 — 李录"文明演进框架"

### 文明级范式转移判断：**是，但时间线不确定**

全球核电正处于**50 年来最大政策拐点**：
- 美国 EO（2025.05）：2050 年核电从 100GW 扩至 **400GW**
- COP29：31 国承诺 2050 年核电容量**三倍**
- AI 数据中心：2026–2030 年新增 50–100GW 基载电力需求（估计，McKinsey/BloombergNEF）
- IRA §45U：零排放核电 PTC 0.3¢/kWh（通胀调整后），可 5 倍放大

### TAM 与天花板

| 来源 | 2025–2026 TAM | 2030s 预测 |
|------|---------------|------------|
| Research and Markets | $8.9B | $12.5B（2034） |
| Coherent Market Insights | $6.3B（2026） | $8.1B（2033） |

**NuScale 可寻址市场**：77 MWe 模块 × $35–55M/模块（PMA 里程碑价）× 全球 SMR 部署量。若 TVA 6GW 落地 = ~78 模块 × $40M = **$31 亿**模块收入（5–10 年分摊）。

### 产业链位置

```
上游：铀矿、燃料（Framatome 合作）
中游：NuScale（设计+模块制造）← 本报告焦点
下游：ENTRA1/Fluor（EPC+融资+运营）→ 公用事业/数据中心
```

NuScale 位于**价值链利润最高但风险最大**的环节——设计/IP 层。但 ENTRA1 独家协议使 NuScale 更像是 ENTRO1 的 OEM 供应商，而非独立平台。

### 李录式追问

> **站在20年后回看，这家公司是"这个时代的标准石油"还是"昙花一现的3Com"？**

**标准石油情景（概率 20%）**：SMR 成为基载电力主流技术，NuScale 设计全球部署 100+ 模块，公司成为"核电界的 ARM"——卖 IP 收版税。

**3Com 情景（概率 40%）**：SMR 行业爆发但 NuScale 因 FOAK 成本失败、ENTRA1 依赖、竞争落后而边缘化，被 TerraPower/GEH 取代。

**Ballard 情景（概率 30%）**：SMR 行业整体商业化慢于预期，NuScale 有证无单，长期烧钱但未破产（靠 ATM 续命）。

**最可能（观点）**：介于 3Com 与 Ballard 之间——行业趋势真实，NuScale 有监管资产，但**当前结构和估值不支持"标准石油"叙事**。
