## 摘要：核心结论先行

| 维度 | 结论 |
|------|------|
| 主题定义 | 中国企业在进口依赖/出口管制约束赛道，以产品性能与客户验证替代海外供应商；**不是单一行业，而是跨设备、材料、设计、工控、医疗、信创的政策+商业双驱动主题** |
| 行业位置 | **扩张期中后段（结构分化）**——设备整体国产化率约 21%–35%（口径不一），成熟环节份额提升已部分定价；光刻/量测仍早期；医疗/工控替代进入全球份额争夺阶段 |
| 终选 3 家 | **迈瑞医疗（核心）/ 汇川技术（卫星）/ 豪威集团（成长）** |
| 组合逻辑 | 医疗器械质量锚 × 工控国产替代成长 × CIS/车载视觉半导体弹性 |
| 最大反直觉发现 | **硬指标下，半导体设备龙头几乎全军覆没**（PE 100×+ 或 OCF/净利不足）；能过价值筛的"国产替代"标的，反而集中在医疗、工控、CIS |
| 最大风险 | 设备/算力主题估值泡沫回撤；迈瑞国内复苏不及预期；汇川新能源业务稀释工控纯度；豪威消费电子周期波动 |
| 主题仓位上限（观点） | 总仓位的 **8–15%**（主题宽、个股 α 分化极大；不想选股用 ETF） |

> 芒格："反过来想，总是反过来想。"——热门赛道里买不到安全边际时，先问：哪里有真实替代、且价格还讲理？

## 第一层：全市场扫描（A ∪ B ∪ C）

### 1.1 主题结构速记

```
国产替代（政策+出口管制+客户验证）
├─ 半导体硬核
│   ├─ 设备：刻蚀/薄膜/清洗/CMP/检测（北方华创、中微、盛美、拓荆、华海清科…）
│   ├─ 材料：硅片/抛光液垫/光刻胶/特气/靶材
│   ├─ 制造：中芯、华虹、长鑫(IPO)、长江存储(辅导)
│   └─ 设计：海光/寒武纪/摩尔线程、澜起、豪威、兆易…
├─ 工业母机/工控：汇川、科德数控、绿的谐波、埃斯顿
├─ 工业软件/信创：华大九天、金山办公、用友、达梦、中国软件
└─ 医疗器械：迈瑞、联影、开立、安图
```

**国产化率事实（需注意口径冲突）**：
- Bernstein：2025 年中国 WFE 整体国产化率 **16%→21%**（国泰基金/一财引用，2026-06）
- 部分中文研报称 2026 已"跨越 30%"；Asia Times 引述称部分前道环节已 50%+，但**光刻 <5%、涂胶显影 <10%、离子注入 <5%**（2026-01）
- **观点**：整体处于"成熟环节份额提升 + 高端瓶颈仍卡"的双速阶段，不是均匀替代

## 第三层：精细分析（10 家）

### 迈瑞医疗（300760）

**一句话商业模式**：向全球医院卖监护、麻醉、超声、IVD 仪器及试剂耗材，设备装机后靠试剂/服务流水变现。

**财务质量**：
- FY25：营收 332.8 亿（-9.4%），归母净利 81.4 亿（-30.3%），ROE 21.6%，毛利率约 60%，OCF/净利 1.25
- 关键转折：国内设备/IVD 承压（国内收入约 -23%），国际收入约 176.5 亿（+7.4%）占比升至 **53%**；Q4 营收已转正

**护城河深度**：多产线规模+装机基础+全球化渠道；5 年后护城河大概率仍在（全球前 30 器械唯一中国公司地位难短期颠覆）。

**主要风险**：1) 国内集采/反腐致资本开支继续低迷；2) 海外地缘与货币风险；3) 高端影像/IVD 与罗氏、西门子等差距。

**估值快评**：PE静 20.9x / PE动 21.7x，相对自身历史与全球器械同业偏合理偏下；一句话：**合理偏低（质量折价因短期利润下滑）**。

**进入终选？**：**是**——全主题唯一接近"巴菲特型"的硬指标满分标的。

### 汇川技术（300124）

**一句话商业模式**：卖变频器、伺服、PLC、电机及新能源动力系统给制造业与新能源汽车客户。

**财务质量**：
- FY25：营收 451.1 亿（+21.8%），归母净利 50.5 亿（+17.8%），ROE 16.3%，OCF 66.8 亿（OCF/净利 1.32），负债率 47.9%
- 毛利率约 29% 企稳；分红预案每 10 股派 5 元

**护城河深度**：工控国产替代份额+产品平台化+渠道；5 年后工控主业护城河大概率仍在，新能源业务竞争更激烈。

### 4.1 迈瑞医疗（300760）— 核心仓

#### 段永平视角：生意本质
卖救命/诊疗设备与试剂给医院——**好生意**：刚需、复购（试剂）、全球可复制。本分是产品力与临床口碑，不是讲故事。2025 利润下滑是国内周期与结构压力，不是模式崩坏；国际占比过半说明已从"国产替代故事"升级为"全球医疗器械竞争者"。管理层连续高分红、未再融资，符合本分。

#### 巴菲特视角：护城河

| 护城河 | 强度 | 具体证据 |
|-------|------|---------|
| 品牌/定价权 | ★★★★ | 国内多品类份额领先；全球前30唯一中国公司 |
| 转换成本 | ★★★★ | 医院装机+试剂耗材绑定；培训与工作流切换成本高 |
| 网络效应 | ★★ | 弱（非平台） |
| 规模效应 | ★★★★★ | 研发分摊、渠道、多产线协同 |
| 技术/牌照壁垒 | ★★★★ | 医疗器械注册与临床验证 |

10 年后护城河大概率仍在。安全边际：PE~21、盈利收益率约 4.8%，相对历史与质量仍有折扣——**但利润尚未见底确认**。

#### 芒格视角：失败模式
1. 国内医院资本开支长期冰冻 + 集采把毛利打穿  
2. 海外政治风险导致欧美高端客户流失  
3. 新兴业务（微创等）并购整合失败、资本回报下降  

最坏情景（极简）：若净利再腰斩至 ~40 亿且给 15x → 市值约 600 亿（现价约腰斩级）。聪明人不买的理由：成长股变成周期股、2025 盈利坍塌未结束。道德/合规风险：医疗反腐与招采合规需持续跟踪，目前无证据显示系统性崩坏。

#### 李录视角：文明级定位
医疗器械国产化与全球可及性提升，更接近**长期结构趋势**而非纯热潮。终局：全球多品类前十常客；不是赢家通吃，但可成稳定复利机器。

#### 综合推荐度
```
推荐度：★★★★★
仓位类型：核心
建议买入区间：当前价可建底仓；若回调至 PE静 16–18x（约对应股价更低一档）可加仓
```
