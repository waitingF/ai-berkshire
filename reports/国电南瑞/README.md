## 一句话结论

国电南瑞是电网二次系统与调度领域的优质龙头——ROE 约 16%、净现金、经营现金流持续覆盖净利润；但定价权偏弱、约六成销售绑定国网体系，且网外/储能扩张正在稀释毛利率。现价静态 PE 约 21x，属「优质公司 + 中等安全边际」，适合观察与分批，而非深度便宜的重仓捡漏。
